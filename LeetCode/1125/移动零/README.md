### LeetCode 第 283 题：移动零 (Move Zeroes)

#### 问题描述
给定一个数组 `nums`，编写一个函数将所有的 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

**示例:**

输入: `[0, 1, 0, 3, 12]`

输出: `[1, 3, 12, 0, 0]`

**说明:**
1. 必须在原数组上操作，不能拷贝额外的数组。
2. 尽量减少操作次数。

#### 用户问题与回答
**用户问题:** `range(len(nums)):` 是用于遍历数组 `nums` 的每个索引。通过 `len(nums)` 获取数组的长度，然后使用 `range()` 生成从 0 到数组长度的索引范围，遍历每个索引以访问数组的所有元素。

**用户问题:** 在代码中的这行：

```python
for i in range(non_zero_index, len(nums)):
```
这部分代码的目的是将数组 `nums` 中从 `non_zero_index` 开始的位置填充为零。

详细解释：
- `non_zero_index` 是指针，指示数组中下一个非零元素应该放置的位置。当遍历完所有元素后，`non_zero_index` 将指向第一个需要被填充为零的位置。
- `range(non_zero_index, len(nums))` 用于遍历从 `non_zero_index` 开始直到数组末尾的所有位置，这些位置在前面的步骤中没有被非零元素覆盖，因此需要将它们设置为零。

这确保了所有零元素都被移动到数组末尾，并且非零元素的相对顺序保持不变。

#### 问题回答
##### 解题思路
使用双指针的方法来解决这个问题。一个指针遍历整个数组，另一个指针记录非零元素的位置。遍历过程中，如果遇到非零元素，就将其移动到前面已经记录的位置，最后再将剩余的位置填上零。

##### 代码实现
```python
def move_zeroes(nums):
    # 初始化指针，表示下一个非零元素要放置的位置
    non_zero_index = 0
    
    # 遍历整个数组
    for i in range(len(nums)):
        # 如果当前元素不为零，则将其移动到 non_zero_index 位置
        if nums[i] != 0:
            nums[non_zero_index] = nums[i]
            non_zero_index += 1
    
    # 将剩余的位置填上零
    for i in range(non_zero_index, len(nums)):
        nums[i] = 0

# 示例用法
nums = [0, 1, 0, 3, 12]
move_zeroes(nums)
print(nums)  # 输出: [1, 3, 12, 0, 0]
```

##### 复杂度分析
- **时间复杂度**: O(n)，其中 n 是数组的长度。数组中的每个元素最多被操作两次（一次检查是否为零，一次移动）。
- **空间复杂度**: O(1)，只使用了常数级别的额外空间。

##### 总结
使用双指针的方法可以高效地将数组中的零移动到末尾，同时保持非零元素的相对顺序。这个方法只需要一次遍历，并且在原地进行修改，符合题目要求。
