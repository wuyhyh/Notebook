## 20250827

1. PBF DDR training失败

解决办法是在 PBF 配置阶段关闭 one channel drive two slot

2. uboot shell 和 ramdisk Linux 找不到 nvme SSD

解决办法是在 PBF 配置阶段配置 PCIe 为 x8x8，RC mode

3. uboot 使用串口工具进行更新
