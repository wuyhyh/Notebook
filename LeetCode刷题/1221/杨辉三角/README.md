LeetCode第118题是《杨辉三角》（Pascal's Triangle），题目要求生成杨辉三角的前 `numRows` 行。

### 题目描述

给定一个整数 `numRows`，返回杨辉三角的前 `numRows` 行。

**杨辉三角**是一个三角形数组，每个数字是上一行的两个数字之和。具体地，第一行和最后一行的数字是 `1`，其他行的数字是通过加法得到的。

### 示例

**示例 1：**

```plaintext
输入: numRows = 5
输出: [
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]
]
```

**示例 2：**

```plaintext
输入: numRows = 1
输出: [
 [1]
]
```

### 解题思路

杨辉三角的构建规则是每个元素等于它上面两个元素之和，边缘元素始终为 1。可以逐行构建杨辉三角的每一行。

#### 思路：
1. **第一行**始终是 `[1]`。
2. 对于**第二行到最后一行**，可以根据上一行的元素来构造：
    - 第一列和最后一列为 `1`。
    - 其他位置的元素是上一行对应位置的两个数之和。

#### 具体步骤：
- 从第一行开始，逐行构建，直到 `numRows` 行。
- 每一行可以通过上一行的元素生成。
- 对于第 `i` 行，长度是 `i + 1`，并且每个元素的值可以通过上一行的元素计算。

### 代码实现

```python
def generate(numRows):
    result = []
    
    # 遍历生成每一行
    for i in range(numRows):
        # 每行的第一个元素和最后一个元素都是 1
        row = [1] * (i + 1)
        
        # 中间的元素是上一行对应元素之和
        for j in range(1, i):
            row[j] = result[i - 1][j - 1] + result[i - 1][j]
        
        result.append(row)
    
    return result
```

### 代码解析

1. **`result = []`**：初始化一个空列表，用来存储杨辉三角的每一行。
2. **`for i in range(numRows):`**：通过循环生成 `numRows` 行杨辉三角。
    - 对于每一行，首先创建一个包含 `i + 1` 个 `1` 的列表 `row`。
    - **`row = [1] * (i + 1)`**：初始化当前行，每个位置都设置为 1。
3. **`for j in range(1, i):`**：对于当前行的中间位置（即除了最前和最后的元素），用上一行的两个相邻元素的和来填充：
    - **`row[j] = result[i - 1][j - 1] + result[i - 1][j]`**：当前行的中间元素是上一行相邻两个元素的和。
4. **`result.append(row)`**：将当前行添加到结果列表 `result` 中。
5. 最终，返回 `result`，即包含 `numRows` 行的杨辉三角。

### 时间复杂度

- **时间复杂度**：`O(n^2)`，其中 `n` 是 `numRows`。每一行的生成都需要遍历上一行并做一定的计算，因此总时间复杂度为 `O(n^2)`。

- **空间复杂度**：`O(n^2)`，需要存储 `numRows` 行的杨辉三角，每行的元素数量是逐渐增多的，因此空间复杂度为 `O(n^2)`。

### 示例解析

#### 示例 1：`numRows = 5`

1. **第一行**：`[1]`
2. **第二行**：`[1, 1]`
3. **第三行**：`[1, 2, 1]`，其中中间元素 `2` 是 `1 + 1`。
4. **第四行**：`[1, 3, 3, 1]`，其中 `3` 是 `1 + 2` 和 `2 + 1`。
5. **第五行**：`[1, 4, 6, 4, 1]`，其中 `4` 是 `1 + 3`，`6` 是 `3 + 3`，`4` 是 `3 + 1`。

最终输出为：

```plaintext
[
 [1],
 [1, 1],
 [1, 2, 1],
 [1, 3, 3, 1],
 [1, 4, 6, 4, 1]
]
```

#### 示例 2：`numRows = 1`

1. **第一行**：`[1]`

输出：

```plaintext
[[1]]
```

### 总结

- **杨辉三角**的构建可以通过逐行构建，每行的元素根据上一行的元素计算得出。
- 利用**动态规划**的思想，可以通过存储每一行的结果来避免重复计算。
- 时间复杂度为 `O(n^2)`，适合较小规模的输入。
