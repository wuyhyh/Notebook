# LeetCode 第 239 题：滑动窗口最大值

## 问题描述

给你一个整数数组 `nums` 和一个滑动窗口的大小 `k`，请你在数组中找到每个滑动窗口中的最大值。

滑动窗口的大小为 `k`，你需要在数组的每个位置上输出一个窗口的最大值。

### 示例：

**输入**:
```plaintext
nums = [1,3,-1,-3,5,3,6,7], k = 3
```

**输出**:
```plaintext
[3,3,5,5,6,7]
```

**解释**:
- 滑动窗口的位置及最大值为：
    - [1 3 -1] -> 3
    - [3 -1 -3] -> 3
    - [-1 -3 5] -> 5
    - [-3 5 3] -> 5
    - [5 3 6] -> 6
    - [3 6 7] -> 7

## 解题思路

使用 **双端队列**（Deque）来维护当前窗口的最大值。队列中的元素按照降序排列，队首始终是当前窗口的最大值。通过双端队列的特性，我们可以高效地解决滑动窗口最大值问题。

### 双端队列的作用

- 队列中存储的是元素的 **索引**，而不是元素的值。通过索引，我们可以快速访问元素。
- 保持队列按 **降序排列**，这样队首元素始终是当前窗口中的最大值。
- 当窗口滑动时，队首的元素会移出队列，因为它不再属于当前窗口。

### 算法步骤

1. **初始化**：
    - 使用一个双端队列 `dq` 来存储索引，确保它按降序排列。
    - 使用 `result` 列表来存储每个窗口的最大值。

2. **处理每个元素**：
    - 对于每个元素，我们首先检查队列尾部的元素是否小于当前元素。如果是，就把队尾的元素移除。
    - 将当前元素的索引加入队列。
    - 如果当前索引已经达到 `k - 1`，则记录当前窗口的最大值（队首元素）。

3. **滑动窗口**：
    - 每次滑动窗口时，队列会自动更新，移除不在当前窗口范围内的元素。

### 代码实现

```python
from collections import deque

def maxSlidingWindow(nums, k):
    # 边界条件：如果数组为空，直接返回空列表
    if not nums:
        return []
    
    # 双端队列，保存的是元素的索引，保证队列按降序排列
    dq = deque()
    result = []
    
    for i in range(len(nums)):
        # 移除队首元素，如果它已经不在当前窗口内
        if dq and dq[0] < i - k + 1:
            dq.popleft()
        
        # 移除队尾的元素，直到队尾的元素大于当前元素
        while dq and nums[dq[-1]] < nums[i]:
            dq.pop()
        
        # 将当前元素的索引加入队列
        dq.append(i)
        
        # 当索引达到 k - 1 时，开始记录窗口最大值
        if i >= k - 1:
            result.append(nums[dq[0]])  # 队首元素是当前窗口的最大值
    
    return result
```

### 代码注释解释

#### 1. **初始化部分**：
```python

    return []
```
- 如果数组为空，直接返回空列表。

```python
dq = deque()  # 双端队列，存储当前窗口的索引
result = []  # 结果列表，用来保存每个窗口的最大值
```
- 使用 `deque` 来存储当前窗口的索引。`result` 用来保存每个窗口的最大值。

#### 2. **遍历数组并更新队列**：
```python
for i in range(len(nums)):
    # 移除队首元素，如果它已经不在当前窗口内
    if dq and dq[0] < i - k + 1:
        dq.popleft()
```
- 对于每个新元素，我们首先检查队首的元素是否已经超出当前窗口。如果超出窗口范围，就把队首元素移除。

```python
# 移除队尾的元素，直到队尾的元素大于当前元素
while dq and nums[dq[-1]] < nums[i]:
    dq.pop()
```
- 我们确保队列中的元素按降序排列。如果当前元素 `nums[i]` 大于队尾元素 `nums[dq[-1]]`，就不断地从队尾移除元素，直到找到一个比当前元素大的元素。

```python
dq.append(i)  # 将当前元素的索引加入队列
```
- 当前元素的索引加入队列。

#### 3. **记录每个窗口的最大值**：
```python
if i >= k - 1:
    result.append(nums[dq[0]])  # 队首元素是当前窗口的最大值
```
- 当我们遍历到索引 `i` 且 `i >= k - 1` 时，说明当前窗口已经有 `k` 个元素，此时队首元素对应的值是当前窗口的最大值，我们将它加入 `result`。

### 时间复杂度分析

- **时间复杂度**：每个元素至多被添加到队列一次，也会被从队列中移除一次。因此，总的时间复杂度为 `O(n)`，其中 `n` 是数组 `nums` 的长度。
- **空间复杂度**：队列最多存储 `k` 个索引，因此空间复杂度是 `O(k)`。

### 结论

使用双端队列来解决滑动窗口问题是一个非常高效的方法，能在 `O(n)` 时间内完成所有操作，确保每次滑动窗口时都能高效地找到最大值。通过队列的降序排列，我们确保了队首元素始终是当前窗口的最大值。
