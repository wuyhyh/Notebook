LeetCode 第543题：“二叉树的直径”（**Diameter of Binary Tree**）要求我们计算给定二叉树的直径。二叉树的直径是树中任意两节点之间的最长路径的长度，路径可以穿过根节点，也可以不穿过根节点。

### 思路：
1. **直径的定义**：
    - 树的直径是两点之间的最长路径。对于任意一个节点，它的直径是其左子树的最大深度加上右子树的最大深度。
    - 然后，对于整个树的直径，我们需要遍历每一个节点，计算每个节点的直径，并返回其中最大的值。

2. **递归解法**：
    - 递归过程中，我们计算每个节点的左右子树的深度，并更新当前的直径。
    - 对于每个节点，直径等于左子树的深度加上右子树的深度。
    - 同时，递归返回当前节点的深度（即，左子树和右子树深度的最大值加1）。

3. **递归实现步骤**：
    - 递归函数返回的是当前节点的最大深度，同时更新直径。
    - 在递归过程中，更新当前最大直径。

### 代码实现：

```python
class Solution(object):
    def diameterOfBinaryTree(self, root):
        """
        :type root: TreeNode
        :rtype: int
        """
        self.diameter = 0  # 用于存储最大直径
        
        # 定义一个递归函数计算深度并更新最大直径
        def depth(node):
            if not node:
                return 0  # 空节点的深度是0
            
            # 递归计算左右子树的深度
            left_depth = depth(node.left)
            right_depth = depth(node.right)
            
            # 更新最大直径
            self.diameter = max(self.diameter, left_depth + right_depth)
            
            # 返回当前节点的深度
            return 1 + max(left_depth, right_depth)
        
        # 从根节点开始递归计算
        depth(root)
        
        # 最终返回的是最大直径
        return self.diameter
```

### 解释：
- **`diameterOfBinaryTree`** 方法：该方法是主函数，它初始化一个 `self.diameter` 变量用于存储当前找到的最大直径。
- **`depth`** 方法：递归计算每个节点的深度，同时在每次递归时更新最大直径。递归函数返回的是当前节点的深度，即它的左子树和右子树的最大深度加1。
- **递归终止条件**：当节点为空时，返回深度 `0`，因为空节点没有深度。
- **更新直径**：对于每个节点，当前节点的直径等于左子树深度加右子树深度。更新全局的最大直径 `self.diameter`。

### 示例：
给定以下二叉树：
```
       1
      / \
     2   3
    / \
   4   5
```

- 节点 `1` 的直径是 `4`（从 `4` 到 `5`，经过 `2`）。
- 节点 `2` 的直径是 `3`（从 `4` 到 `5`）。
- 节点 `3` 的直径是 `0`（没有子节点）。
- 最终，树的直径是 `4`。

### 时间复杂度：
- **时间复杂度**：\(O(n)\)，其中 `n` 是树的节点数。我们遍历每个节点一次。
- **空间复杂度**：\(O(h)\)，其中 `h` 是树的高度。递归栈的空间复杂度是树的高度，在最坏情况下（树是链状的）空间复杂度为 \(O(n)\)，在最好的情况下（树是平衡的）空间复杂度为 \(O(\log n)\)。

### 总结：
- **递归解法**：使用递归计算每个节点的深度，并更新最大直径。该方法时间复杂度为 \(O(n)\)，空间复杂度为 \(O(h)\)（递归栈深度）。这种解法简洁而高效，适用于绝大多数二叉树问题。
