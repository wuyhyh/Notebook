LeetCode第279题是《完全平方数》（Perfect Squares），题目要求给定一个正整数 `n`，找到若干个完全平方数的和，使得它们的和等于 `n`。你需要返回最少需要多少个完全平方数来表示 `n`。

### 题目描述

给定一个整数 `n`，返回总和为 `n` 的完全平方数的最少数量。

**完全平方数** 是指一个整数是另一个整数的平方，例如 `1, 4, 9, 16, 25, 36, ...`。

### 示例

**示例 1：**

```plaintext
输入: n = 12
输出: 3
解释: 12 = 4 + 4 + 4
```

**示例 2：**

```plaintext
输入: n = 13
输出: 2
解释: 13 = 4 + 9
```

### 解题思路

这是一个典型的**动态规划**问题。我们可以通过构建一个动态规划数组 `dp` 来表示从 `1` 到 `n` 的每个数最少需要多少个完全平方数。

#### 思路：
1. **状态定义**：
    - 定义 `dp[i]` 为表示整数 `i` 的最少完全平方数的数量。

2. **状态转移**：
    - 对于每个 `i`，我们可以尝试用一个完全平方数 `j*j` 来表示 `i`，然后从 `i-j*j` 得到答案。这样，`dp[i]` 就是 `dp[i-j*j] + 1` 的最小值，其中 `j*j` 是一个完全平方数，`j*j <= i`。

   所以，我们的状态转移方程是：
   \[
   dp[i] = \min(dp[i], dp[i - j*j] + 1) \quad \text{for all } j \text{ such that } j*j \leq i
   \]

3. **初始条件**：
    - `dp[0] = 0`：表示数字 `0` 需要 `0` 个完全平方数。
    - 对于 `i >= 1`，初始值 `dp[i]` 可以设置为 `i`，因为最坏的情况下，`i` 可以表示为 `i` 个 `1`（完全平方数）。

4. **结果**：
    - 最终，`dp[n]` 就是我们要求的最小完全平方数的个数。

### 代码实现

```python
import math

def numSquares(n):
    # 初始化dp数组，dp[i]表示i的最少完全平方数的个数
    dp = [float('inf')] * (n + 1)
    dp[0] = 0  # 0 需要 0 个完全平方数

    # 填充dp数组
    for i in range(1, n + 1):
        j = 1
        while j * j <= i:
            dp[i] = min(dp[i], dp[i - j * j] + 1)
            j += 1

    return dp[n]
```

### 代码解析

1. **初始化 `dp` 数组**：
    - `dp[i]` 表示 `i` 需要的最少完全平方数的个数。初始化时，将 `dp` 数组的所有元素设为一个很大的值 `float('inf')`，表示还没有计算出来。然后，将 `dp[0] = 0`，因为 `0` 的最少完全平方数是 `0`。

2. **遍历每个数字 `i` 从 `1` 到 `n`**：
    - 对于每个数字 `i`，我们通过循环尝试每个可能的完全平方数 `j*j`（其中 `j*j <= i`）。
    - 对于每个 `i`，检查所有小于等于 `i` 的完全平方数 `j*j`，计算 `dp[i] = min(dp[i], dp[i - j*j] + 1)`。即通过使用 `j*j` 来更新 `dp[i]`，选择最小的值。

3. **返回结果**：
    - `dp[n]` 就是答案，即表示数字 `n` 所需要的最少完全平方数的个数。

### 时间复杂度

- **时间复杂度**：`O(n * sqrt(n))`，其中 `n` 是输入数字。对于每个 `i`，我们需要遍历所有小于等于 `sqrt(i)` 的完全平方数，即循环次数为 `sqrt(i)`，总的时间复杂度为 `O(n * sqrt(n))`。

- **空间复杂度**：`O(n)`，需要一个大小为 `n+1` 的数组来存储每个数字的最少完全平方数个数。

### 总结

- **动态规划** 解决了从小到大的递推问题，逐步计算出每个数字所需要的最少完全平方数的个数。
