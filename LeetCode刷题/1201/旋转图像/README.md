第48题是“旋转图像”（Rotate Image），要求将一个 `n x n` 的矩阵顺时针旋转 90 度。

### 问题描述

给定一个 `n x n` 的矩阵，原地将矩阵旋转 90 度。

**注意：**
- 你必须在 **原地** 旋转矩阵，也就是说，不能使用额外的矩阵来存储旋转后的结果。

### 示例

**示例 1:**

输入：
```plaintext
[
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]
```

输出：
```plaintext
[
  [7, 4, 1],
  [8, 5, 2],
  [9, 6, 3]
]
```

**示例 2:**

输入：
```plaintext
[
  [5, 1, 9, 11],
  [2, 4, 8, 10],
  [13, 3, 6, 12],
  [14, 15, 7, 16]
]
```

输出：
```plaintext
[
  [14, 13, 2, 5],
  [15, 3, 4, 1],
  [7, 6, 8, 9],
  [16, 12, 10, 11]
]
```

## 解题思路

### 方案：先转置，再翻转

我们可以通过 **两次操作** 来完成旋转：

1. **转置矩阵**：将矩阵的行和列互换。即矩阵的第 `i` 行第 `j` 列的元素，变为第 `j` 行第 `i` 列的元素。
2. **水平翻转**：将转置后的矩阵进行水平翻转（即将每行的元素顺序颠倒）。

### 具体步骤

1. **转置矩阵**：将矩阵的每个元素 `matrix[i][j]` 与 `matrix[j][i]` 交换。
2. **水平翻转**：对矩阵的每一行进行翻转。

#### 代码实现：

```python
def rotate(matrix):
    n = len(matrix)
    
    # 1. 转置矩阵
    for i in range(n):
        for j in range(i + 1, n):
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
    
    # 2. 水平翻转每一行
    for i in range(n):
        matrix[i].reverse()
```

### 代码解释

1. **转置矩阵**：  
   外层循环 `i` 遍历每一行，内层循环 `j` 遍历每行的元素（`j` 从 `i + 1` 开始，避免重复交换）。在这里，我们将 `matrix[i][j]` 与 `matrix[j][i]` 交换，完成矩阵转置。

2. **水平翻转每一行**：  
   对每一行使用 `.reverse()` 方法进行翻转。这样，矩阵就顺时针旋转了 90 度。

### 时间复杂度和空间复杂度

- **时间复杂度**：`O(n^2)`，由于我们遍历矩阵中的每个元素进行转置和翻转操作，时间复杂度是 `O(n^2)`，其中 `n` 是矩阵的大小。
- **空间复杂度**：`O(1)`，我们没有使用额外的空间，只在原地修改矩阵，因此空间复杂度为常数 `O(1)`。

## 总结

1. **转置矩阵**：通过交换 `matrix[i][j]` 和 `matrix[j][i]` 实现矩阵转置。
2. **水平翻转**：对每一行进行翻转，完成旋转 90 度。
