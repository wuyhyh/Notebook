第234题是“回文链表”（Palindrome Linked List），要求判断一个单链表是否是回文链表。

### 问题描述

给定一个单链表的头节点 `head`，判断链表是否为回文链表。回文链表指的是正着读和倒着读都相同的链表。

### 示例

**示例 1:**

输入：
```plaintext
head = [1, 2, 2, 1]
```

输出：
```plaintext
True
```

**示例 2:**

输入：
```plaintext
head = [1, 2]
```

输出：
```plaintext
False
```

### 解题思路

判断一个链表是否为回文链表，最直接的方式是将链表中的元素放入一个数组或栈中，利用其顺序读取和反向读取来判断回文，但这需要额外的空间。

我们可以使用 **快慢指针** 和 **反转链表** 的方法来 **原地** 判断链表是否是回文链表。具体步骤如下：

### 思路

1. **使用快慢指针**：
    - 通过快慢指针找到链表的中点。慢指针一次走一步，快指针一次走两步。快指针到达链表末尾时，慢指针正好到达链表的中点。

2. **反转后半部分链表**：
    - 将链表的后半部分进行反转。这样我们可以将前半部分和反转后的后半部分逐一进行比较。

3. **比较前后两部分链表**：
    - 比较前半部分和反转后的后半部分是否相等。如果相等，则链表是回文的，返回 `True`；否则，返回 `False`。

4. **恢复链表**：
    - 反转后的链表部分可以选择恢复成原来的顺序，保证链表结构的完整性，但这个步骤不是必须的。

### 代码实现

```python
# Definition for singly-linked list.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def isPalindrome(head: ListNode) -> bool:
    if not head or not head.next:
        return True
    
    # 快慢指针找到链表的中点
    slow, fast = head, head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    
    # 反转后半部分链表
    prev = None
    while slow:
        next_node = slow.next
        slow.next = prev
        prev = slow
        slow = next_node
    
    # 比较前半部分和后半部分
    left, right = head, prev
    while right:  # 只需要比较后半部分
        if left.val != right.val:
            return False
        left = left.next
        right = right.next
    
    return True
```

### 代码解释

1. **边界情况**：
    - 如果链表为空或只有一个节点，直接返回 `True`，因为空链表或只有一个节点的链表肯定是回文的。

2. **快慢指针找到中点**：
    - 使用两个指针 `slow` 和 `fast`，`slow` 每次走一步，`fast` 每次走两步。当 `fast` 到达链表末尾时，`slow` 恰好到达链表的中点。

3. **反转链表的后半部分**：
    - 使用 `prev` 和 `slow` 指针反转链表的后半部分，`slow` 最终指向链表的尾部，`prev` 指向反转后的头部。

4. **比较前后两部分链表**：
    - 使用两个指针 `left` 和 `right`，分别指向链表的前半部分和反转后的后半部分，逐一比较它们的值。如果有任何不相等的节点，返回 `False`，表示链表不是回文的。

5. **返回结果**：
    - 如果比较完所有节点都相等，返回 `True`，表示链表是回文的。

### 时间复杂度和空间复杂度

- **时间复杂度**：`O(n)`，其中 `n` 是链表的长度。我们需要遍历链表两次：一次找到中点，一次比较前后两部分。
- **空间复杂度**：`O(1)`，我们只用了常数的空间来存储指针，链表本身没有额外的空间消耗。

### 总结

1. **快慢指针**：使用快慢指针找到链表的中点。
2. **反转后半部分**：反转链表的后半部分，以便进行比较。
3. **比较**：比较链表的前半部分和反转后的后半部分。
4. **时间复杂度**：`O(n)`，空间复杂度：`O(1)`。
