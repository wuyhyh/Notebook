LeetCode 第108题：“将有序数组转换为二叉搜索树”（**Convert Sorted Array to Binary Search Tree**）要求将一个递增排序的数组转换为一棵高度平衡的二叉搜索树（BST）。

### 思路：
1. **二叉搜索树的性质**：
    - 左子树的所有节点值小于根节点值；
    - 右子树的所有节点值大于根节点值；
    - 二叉树的高度平衡意味着每个节点的左右子树高度差不超过 1。

2. **分治法**：
    - 递归选择数组的中间元素作为根节点。
    - 中间元素的左侧部分用于构造左子树，右侧部分用于构造右子树。
    - 递归地对左右子数组重复此过程，直到数组为空。

3. **保证高度平衡**：
    - 通过每次选择中间元素作为根节点，确保左右子树的大小尽可能相等，从而构建出高度平衡的二叉搜索树。

### 代码实现：

```python
class TreeNode(object):
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution(object):
    def sortedArrayToBST(self, nums):
        """
        :type nums: List[int]
        :rtype: TreeNode
        """
        # 边界条件：如果数组为空，返回 None
        if not nums:
            return None
        
        # 选择中间元素作为根节点
        mid = len(nums) // 2
        root = TreeNode(nums[mid])
        
        # 递归构造左子树和右子树
        root.left = self.sortedArrayToBST(nums[:mid])  # 左部分构造左子树
        root.right = self.sortedArrayToBST(nums[mid+1:])  # 右部分构造右子树
        
        return root
```

### 解释：
1. **数组分割**：
    - 每次递归选择数组的中间元素 `nums[mid]` 作为当前子树的根节点。
    - 左侧数组 `nums[:mid]` 用于构造左子树，右侧数组 `nums[mid+1:]` 用于构造右子树。

2. **递归终止条件**：
    - 当数组为空时，返回 `None`，表示当前子树为空。

3. **构造过程**：
    - 每次递归都会创建一个 `TreeNode` 对象，并递归地构造它的左右子树。

### 示例：

输入：
```python
nums = [-10, -3, 0, 5, 9]
```

输出的二叉搜索树可能是：
```
      0
     / \
   -3   9
   /   /
 -10  5
```

### 递归过程：
1. 数组 `[-10, -3, 0, 5, 9]`，选择中间元素 `0` 作为根节点。
2. 左侧数组 `[-10, -3]` 构造左子树，右侧数组 `[5, 9]` 构造右子树。
3. 递归对 `[-10, -3]`：
    - 选择 `-3` 作为左子树的根节点，`[-10]` 构造它的左子树。
4. 递归对 `[5, 9]`：
    - 选择 `9` 作为右子树的根节点，`[5]` 构造它的左子树。

### 时间复杂度：
- **时间复杂度**：\(O(n)\)，其中 `n` 是数组的长度。每个元素都会被访问一次，用于创建节点。
- **空间复杂度**：\(O(\log n)\)，由于递归栈的深度等于树的高度，而平衡二叉搜索树的高度是 \(O(\log n)\)。

### 总结：
- **递归分治法**是最适合这道题的方法，简单且高效。
- 递归地选择中间元素作为根节点，可以保证构造出的树是高度平衡的。
