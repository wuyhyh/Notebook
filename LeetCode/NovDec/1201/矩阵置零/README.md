# LeetCode 73. 矩阵置零（Set Matrix Zeroes）

## 问题描述

给定一个 `m x n` 的矩阵，如果其中某个元素为零，则将该元素所在的整行和整列都置为零。

### 示例

**示例 1:**

输入：

```plaintext
[
  [1, 1, 1],
  [1, 0, 1],
  [1, 1, 1]
]
```

输出：

```plaintext
[
  [1, 0, 1],
  [0, 0, 0],
  [1, 0, 1]
]
```

**示例 2:**

输入：

```plaintext
[
  [0, 1, 2, 0],
  [3, 4, 5, 2],
  [1, 3, 1, 5]
]
```

输出：

```plaintext
[
  [0, 0, 0, 0],
  [0, 4, 5, 0],
  [0, 3, 1, 0]
]
```

## 解题思路

### 方案 1：原地修改（标记法）

我们可以利用矩阵的 **第一行** 和 **第一列** 来标记哪些行和列需要置零。此方法的优势是 **不需要额外的空间**，仅利用原矩阵本身来保存标记信息。

#### 步骤：

1. **判断第一行和第一列是否需要置零**  
   首先检查矩阵的第一行和第一列是否包含零，因为如果它们含有零，我们不能使用它们来标记其他行列。

2. **遍历矩阵并标记**  
   从第二行和第二列开始遍历矩阵。对于每个元素，如果遇到零，则将该元素所在行的第一列和该元素所在列的第一行的对应位置设置为零。

3. **修改矩阵**  
   根据标记，将相应的元素置为零。遍历矩阵，对于每个元素，如果它所在的行或列对应的首元素是零，则将该元素置为零。

4. **处理第一行和第一列**  
   最后，根据第一行和第一列的标记，分别决定是否需要将它们置为零。

#### 代码实现：

```python
def setZeroes(matrix):
    # 记录第一行和第一列是否需要置零
    first_row_zero = any(matrix[0][j] == 0 for j in range(len(matrix[0])))
    first_col_zero = any(matrix[i][0] == 0 for i in range(len(matrix)))
    
    # 使用第一行和第一列作为标记
    for i in range(1, len(matrix)):  # 从第二行开始
        for j in range(1, len(matrix[0])):  # 从第二列开始
            if matrix[i][j] == 0:
                matrix[i][0] = 0  # 标记第i行
                matrix[0][j] = 0  # 标记第j列

    # 根据标记将相应的元素置为零
    for i in range(1, len(matrix)):
        for j in range(1, len(matrix[0])):
            if matrix[i][0] == 0 or matrix[0][j] == 0:
                matrix[i][j] = 0
    
    # 处理第一行
    if first_row_zero:
        for j in range(len(matrix[0])):
            matrix[0][j] = 0

    # 处理第一列
    if first_col_zero:
        for i in range(len(matrix)):
            matrix[i][0] = 0
```

## 时间和空间复杂度分析

- **时间复杂度**：`O(m * n)`  
  我们遍历了矩阵多次，每次遍历的时间是 `O(m * n)`，其中 `m` 是矩阵的行数，`n` 是列数。

- **空间复杂度**：`O(1)`  
  由于我们没有使用额外的二维数组，只使用了常数空间来保存 `first_row_zero` 和 `first_col_zero` 标记，因此空间复杂度为常数空间 `O(1)`。
