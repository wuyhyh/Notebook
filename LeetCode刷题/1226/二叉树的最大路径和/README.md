LeetCode第124题是《二叉树中的最大路径和》（Binary Tree Maximum Path Sum），题目要求计算二叉树中的最大路径和。路径可以从任意节点开始，沿着父子节点方向往下走，可以选择向左或向右走。

### 题目描述

给定一个非空二叉树，返回其最大路径和。路径的定义为：路径可以从任意节点开始，沿着父子节点方向往下走。每个节点的路径和等于该节点的值加上左右子树的最大路径和。每个节点最多只能被访问一次。

#### 示例 1：

```plaintext
输入：
       1
      / \
     2   3
输出：6
解释：最大路径和为 2 + 1 + 3 = 6。
```

#### 示例 2：

```plaintext
输入：
      -10
      /  \
     9   20
        /  \
       15   7
输出：42
解释：最大路径和为 15 + 20 + 7 = 42。
```

### 解题思路

这道题的核心在于通过递归来计算每个节点为根的子树的最大路径和，并通过一个全局变量来记录最大路径和。

#### 递归思路

1. **节点的最大路径和**：对于每个节点，存在两种选择：
    - 节点值加上左子树的最大路径和；
    - 节点值加上右子树的最大路径和。

   需要选择其中较大的路径，并返回这个值，作为上层节点的贡献。

2. **考虑全局最大路径和**：每个节点的路径和不仅可以经过该节点到其左右子树，还可以作为一个分支路径连接左右子树。因此，对于每个节点，需要计算：
    - 该节点作为路径根时，包含左子树和右子树的路径和。
    - 更新全局最大路径和。

3. **状态转移**：
    - 递归计算左右子树的最大路径和。
    - 返回当前节点作为路径起点的最大贡献（即，节点值 + 左右子树的最大路径和）。

4. **递归函数**：
    - 对于每个节点，递归求解其左右子树的最大路径和。
    - 计算当前节点的最大路径和，并更新全局最大值。
    - 返回当前节点的贡献（即，当前节点值 + 左右子树中的较大路径）。

### 代码实现

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def maxPathSum(root):
    # 用于记录全局最大路径和
    max_sum = float('-inf')

    # 定义递归函数，返回从当前节点出发的最大路径和
    def helper(node):
        nonlocal max_sum
        if not node:
            return 0

        # 计算左子树和右子树的最大路径和，负数要取0，因为不能让路径和变差
        left = max(helper(node.left), 0)  # 左子树最大路径和
        right = max(helper(node.right), 0)  # 右子树最大路径和

        # 更新全局最大路径和
        max_sum = max(max_sum, node.val + left + right)

        # 返回当前节点的最大贡献（即该节点的值 + 左右子树中较大的路径）
        return node.val + max(left, right)

    # 启动递归
    helper(root)

    return max_sum
```

### 代码解析

1. **TreeNode 类**：定义了二叉树的节点结构，其中每个节点包含一个整数值和指向左右子节点的指针。

2. **maxPathSum 函数**：计算给定二叉树的最大路径和。
    - `max_sum` 用于记录全局的最大路径和，初始值设为负无穷，保证即使所有节点值都为负，算法也能正确返回最大路径和。

3. **helper 函数**：这是一个递归函数，计算从当前节点出发的最大路径和。
    - `helper(node)`：如果当前节点为 `None`，返回 0，表示路径和为 0。
    - 计算左右子树的最大路径和：`left` 和 `right`，如果它们是负数，说明应该忽略该子树，因此 `max(helper(node.left), 0)` 和 `max(helper(node.right), 0)` 会忽略负值。
    - 更新全局最大路径和：`max_sum = max(max_sum, node.val + left + right)`。
    - 返回当前节点的最大贡献：`node.val + max(left, right)`，即返回当前节点值加上左或右子树的较大路径和。

4. **返回最大路径和**：最终返回 `max_sum`，即全局的最大路径和。

### 时间复杂度

- **时间复杂度**：`O(n)`，其中 `n` 是二叉树的节点数。每个节点只访问一次。
- **空间复杂度**：`O(h)`，其中 `h` 是树的高度。递归调用栈的空间复杂度与树的高度成正比。

### 示例解析

#### 示例 1：

```plaintext
输入：
       1
      / \
     2   3
输出：6
```

1. 从节点 `2` 向下的最大路径和是 `2`（没有子树）。
2. 从节点 `3` 向下的最大路径和是 `3`（没有子树）。
3. 节点 `1` 的最大路径和是 `1 + 2 + 3 = 6`。
4. 返回结果是 `6`。

#### 示例 2：

```plaintext
输入：
      -10
      /  \
     9   20
        /  \
       15   7
输出：42
```

1. 从节点 `9` 向下的最大路径和是 `9`。
2. 从节点 `15` 向下的最大路径和是 `15`。
3. 从节点 `7` 向下的最大路径和是 `7`。
4. 从节点 `20` 向下的最大路径和是 `20 + 15 + 7 = 42`。
5. 节点 `-10` 的最大路径和是 `-10 + 9 + 42 = 42`。
6. 返回结果是 `42`。

### 总结

| 方法           | 时间复杂度 | 空间复杂度 | 适用场景 |
|----------------|------------|------------|----------|
| 递归 + 回溯法  | `O(n)`     | `O(h)`     | 适用于计算树的路径问题，能有效利用递归的性质求解路径和问题。 |

该解法通过递归计算每个节点的最大贡献，并通过全局变量记录最大路径和，适用于大部分二叉树路径问题。
