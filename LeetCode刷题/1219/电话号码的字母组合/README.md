LeetCode第17题是《电话号码的字母组合》（Letter Combinations of a Phone Number），题目要求给定一个数字字符串，返回所有可能的字母组合，这些字母组合是由数字对应的字母映射表生成的。

### 题目描述

给定一个仅包含数字 `2-9` 的字符串，返回所有它能表示的字母组合。答案可以按 **任意顺序** 返回。

给定数字到字母的映射如下（与电话按键相同）：

```
2 -> "abc"
3 -> "def"
4 -> "ghi"
5 -> "jkl"
6 -> "mno"
7 -> "pqrs"
8 -> "tuv"
9 -> "wxyz"
```

**示例 1：**

```plaintext
输入: digits = "23"
输出: ["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"]
```

**示例 2：**

```plaintext
输入: digits = ""
输出: []
```

**示例 3：**

```plaintext
输入: digits = "2"
输出: ["a", "b", "c"]
```

### 解题思路

这个问题可以通过 **回溯算法** 来解决，回溯算法适用于生成所有可能的组合问题。

#### 思路：
1. **每个数字对应多个字母**，比如数字 `2` 对应 `"abc"`，数字 `3` 对应 `"def"` 等。
2. **递归地生成所有可能的字母组合**。每次选择一个字母进行递归，直到处理完所有数字。

#### 具体步骤：
1. **映射表**：将每个数字映射到其对应的字母。
2. **回溯**：递归地构建每一种字母组合，从第一个数字开始，逐步选择每个数字对应的字母，直到所有数字都被处理完。
3. **终止条件**：当当前字母组合的长度等于输入的数字字符串的长度时，表示一个字母组合完成，加入结果列表。

---

### 代码实现（Python）

```python
def letterCombinations(digits):
    if not digits:
        return []
    
    # 映射每个数字到相应的字母
    digit_to_char = {
        "2": "abc", "3": "def", "4": "ghi", "5": "jkl", 
        "6": "mno", "7": "pqrs", "8": "tuv", "9": "wxyz"
    }
    
    result = []
    
    def backtrack(index, current_combination):
        # 如果当前字母组合的长度与输入的数字字符串长度一致，加入结果
        if index == len(digits):
            result.append(current_combination)
            return
        
        # 获取当前数字对应的字母集
        current_digit = digits[index]
        for char in digit_to_char[current_digit]:
            backtrack(index + 1, current_combination + char)
    
    # 从第一个数字开始回溯
    backtrack(0, "")
    
    return result
```

### 代码解析

1. **`digit_to_char` 映射表**：将数字 `2-9` 映射到对应的字母串。
2. **`backtrack(index, current_combination)`**：
    - `index` 表示当前处理到 `digits` 字符串的哪个位置。
    - `current_combination` 存储当前的字母组合。
    - 当 `index` 等于 `digits` 字符串的长度时，表示当前字母组合完成，将其加入 `result`。
    - 否则，遍历当前数字对应的字母，递归调用 `backtrack`，继续生成下一个字母。
3. **起始调用**：从第一个数字开始回溯，初始化为 `backtrack(0, "")`。

### 时间复杂度

- **时间复杂度**：`O(4^N)`，其中 `N` 是输入的数字字符串的长度。因为每个数字最多对应4个字母（例如数字 `7` 对应 `"pqrs"`），递归过程中每次都要遍历当前数字对应的字母，因此时间复杂度为 `4^N`。
- **空间复杂度**：`O(N)`，递归的栈深度为 `N`，并且结果列表 `result` 的大小也与字母组合的数量成正比。

### 例子解析

以 `digits = "23"` 为例：

1. **初始**：调用 `backtrack(0, "")`，表示从第一个数字开始生成字母组合。
2. **处理数字 `2`**：对应字母 `"abc"`，尝试每个字母：
    - 对于字母 `'a'`，调用 `backtrack(1, "a")`，继续处理数字 `3`。
    - 对于字母 `'b'`，调用 `backtrack(1, "b")`，继续处理数字 `3`。
    - 对于字母 `'c'`，调用 `backtrack(1, "c")`，继续处理数字 `3`。
3. **处理数字 `3`**：对于每个字母，尝试 `'d'`, `'e'`, `'f'`，继续构建字母组合。
    - 例如，`backtrack(1, "a")` 处理时，将会生成字母组合 `"ad"`, `"ae"`, `"af"`，并将它们加入 `result` 中。
4. **终止条件**：当所有数字都处理完时，返回所有的字母组合。

最终返回：

```plaintext
["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"]
```

### 总结

- 这道题通过回溯算法生成所有字母组合，每次递归选择一个字母并继续构建字母组合。
- 回溯算法的递归结构使得问题的解决方式自然地适用于这种生成所有组合的问题。
