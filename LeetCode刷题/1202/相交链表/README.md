第160题是“相交链表”（Intersection of Two Linked Lists），要求找到两个单链表的交点。如果两个链表没有交点，返回 `null`。

### 问题描述

给定两个单链表 `headA` 和 `headB`，如果两个链表相交，则返回相交节点。否则，返回 `null`。

**注意：**
- `headA` 和 `headB` 均为两个链表的头节点。
- 你必须实现一个 **时间复杂度为 O(m + n)**，其中 `m` 和 `n` 分别是两个链表的长度的算法。

### 示例

**示例 1:**

输入：
```plaintext
A:      4 -> 1 ->
                \
                 8 -> 4 -> 5
B:  5 -> 0 -> 1 ->
                \
                 8 -> 4 -> 5
```

输出：
```plaintext
8
```

**示例 2:**

输入：
```plaintext
A:      2 -> 6 -> 4
B:  1 -> 5
```

输出：
```plaintext
null
```

### 解题思路

#### 方案：双指针法

由于链表的长度可能不同，我们不能直接同步遍历两个链表。但是可以利用一个巧妙的技巧，通过同时遍历两个链表并调整指针位置来找到交点。

1. **双指针法**：
    - 定义两个指针 `pA` 和 `pB`，分别指向链表 `A` 和 `B` 的头节点。
    - 将 `pA` 和 `pB` 同时遍历链表。当 `pA` 到达链表 `A` 的尾部时，将其重定向到链表 `B` 的头部；同理，当 `pB` 到达链表 `B` 的尾部时，将其重定向到链表 `A` 的头部。
    - 这样一来，两个指针就会在同等长度的情况下遍历两个链表。每个指针会遍历两个链表的总长度（`m + n`），并且在相交的节点会相遇。

2. **为什么这种方法有效？**
    - 如果链表相交，两个指针最终会在相交节点处相遇。通过让两个指针分别遍历两个链表，它们将经历相同的路径，因此在第二轮遍历中，它们会在交点处相遇。
    - 如果链表不相交，两个指针会在遍历完两个链表后都指向 `null`，这时返回 `null`。

### 代码实现：

```python
# Definition for singly-linked list.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def getIntersectionNode(headA: ListNode, headB: ListNode) -> ListNode:
    # 如果两个链表为空，直接返回 None
    if not headA or not headB:
        return None
    
    # 定义两个指针
    pA, pB = headA, headB
    
    # 遍历两个链表
    while pA != pB:
        # 当指针 pA 到达链表 A 的末尾时，将它指向链表 B 的头部
        # 同理，当指针 pB 到达链表 B 的末尾时，将它指向链表 A 的头部
        pA = pA.next if pA else headB
        pB = pB.next if pB else headA
    
    # 如果有交点，pA 和 pB 会相遇；如果没有交点，pA 和 pB 会同时指向 None
    return pA
```

### 代码解释

1. **初始化**：
    - 创建两个指针 `pA` 和 `pB`，分别指向链表 `A` 和 `B` 的头节点。

2. **双指针遍历**：
    - 在 `while pA != pB` 循环中，两个指针同时遍历两个链表。
    - 当 `pA` 到达链表 `A` 的末尾时，将它指向链表 `B` 的头部，`pB` 也是如此。这样两个指针在第二次遍历时，会在相交点相遇（如果有交点）。

3. **相交节点**：
    - 如果两个链表相交，两个指针最终会在相交节点处相遇。
    - 如果两个链表不相交，则最终两个指针会同时指向 `None`，返回 `None`。

### 时间复杂度和空间复杂度

- **时间复杂度**：`O(m + n)`，其中 `m` 和 `n` 分别是两个链表的长度。每个指针最多会遍历两个链表的总长度一次。
- **空间复杂度**：`O(1)`，只使用了常数的空间来存储指针。

## 总结

1. 使用 **双指针法** 来遍历两个链表，通过调整指针位置，确保两个指针遍历相同的路径，最终在相交点相遇。
2. 如果链表不相交，返回 `None`，如果相交，则返回相交节点。

这种方法是最优解，时间复杂度和空间复杂度都非常高效。
