### **LeetCode 第739题 - 每日温度 (Daily Temperature)**

#### **题目描述**

请根据每日温度的变化情况，返回一个列表 `answer`，其中 `answer[i]` 是指第 `i` 天距离下一个更高温度的天数。如果不存在下一个更高温度，则 `answer[i]` 为 0。

#### **示例**

示例 1:
```python
temperatures = [73, 74, 75, 71, 69, 72, 76, 73]
print(dailyTemperatures(temperatures))  # 输出 [1, 1, 4, 2, 1, 1, 0, 0]
```

示例 2:
```python
temperatures = [30, 40, 50, 60]
print(dailyTemperatures(temperatures))  # 输出 [1, 1, 1, 0]
```

#### **思路分析**

这道题目考察的是如何高效地处理每日温度和其下一个较高温度之间的差距。我们可以使用 **栈** 来解决这个问题。

**思路**：
- 从左到右遍历温度列表，逐个处理每一天的温度。
- 使用栈来记录尚未找到下一个较高温度的天数。栈中的元素是索引值，代表着还没有找到更高温度的天数。
- 对于每一天的温度，我们检查栈顶的元素（表示之前的某一天的温度），如果当前温度比栈顶对应的温度高，那么就找到了栈顶对应的天数的下一个更高温度，将其从栈中弹出，并记录结果。
- 继续将当前天数的索引压入栈中。

这样，栈的作用就是帮助我们记录“尚未找到更高温度的天数”，一旦找到了下一个更高温度，就可以立即得到答案。

### **步骤**：
1. **初始化**：
    - 创建一个空栈 `stack`。
    - 创建一个结果数组 `answer`，大小与输入数组 `temperatures` 相同，初始值全部为 0。

2. **遍历温度数组**：
    - 对于每个温度，检查栈顶元素是否代表着一个温度较低的日子（即当前温度大于栈顶元素所表示的温度）。
    - 如果是，弹出栈顶元素，并记录从栈顶对应的天数到当前天数的天数差。
    - 将当前天数的索引压入栈中。

3. **返回结果**：
    - 最终结果数组 `answer` 中，包含了每一天距离下一个更高温度的天数。

### **代码实现**

```python
def dailyTemperatures(temperatures):
    n = len(temperatures)
    answer = [0] * n  # 初始化答案数组
    stack = []  # 用栈来存储索引
    
    for i in range(n):
        # 当栈不为空且当前温度大于栈顶所表示的温度
        while stack and temperatures[i] > temperatures[stack[-1]]:
            index = stack.pop()  # 弹出栈顶元素，得到前面那个温度较低的天数索引
            answer[index] = i - index  # 计算从那个天数到当前天数的天数差
        stack.append(i)  # 将当前天数索引压入栈中
    
    return answer
```

### **代码解析**

1. **`answer` 初始化**：
    - 我们创建一个和输入 `temperatures` 长度相同的数组 `answer`，用来存储每一天的结果，初始值全为 0。

2. **栈的使用**：
    - 使用一个栈来记录还没有找到下一个较高温度的天数的索引。
    - 遍历温度数组时，我们依次将每一天的索引压入栈中。如果当前温度比栈顶的温度高，表示我们找到了下一个较高温度的天数，弹出栈顶索引，更新结果。

3. **计算天数差**：
    - 当栈顶的温度较低时，我们可以计算出从该天到当前天的天数差，这就是 `answer[index] = i - index`。

4. **返回结果**：
    - 遍历完温度数组后，`answer` 数组中存储的就是每一天距离下一个更高温度的天数。

### **时间复杂度**

- **时间复杂度**：`O(n)`，其中 `n` 是温度数组的长度。每个元素最多被压入栈一次，并且最多被弹出一次，因此时间复杂度为 `O(n)`。
- **空间复杂度**：`O(n)`，我们使用了一个栈来存储索引，空间复杂度为 `O(n)`。

### **示例**

#### 示例 1：

```python
temperatures = [73, 74, 75, 71, 69, 72, 76, 73]
print(dailyTemperatures(temperatures))  # 输出 [1, 1, 4, 2, 1, 1, 0, 0]
```

**解释**：
- 第一天温度是 `73`，第二天温度是 `74`，因此距离下一个更高温度是 1 天。
- 第二天温度是 `74`，第三天温度是 `75`，因此距离下一个更高温度是 1 天。
- 第三天温度是 `75`，第四天温度是 `71`，没有更高的温度，跳过。
- 如此类推，最终返回结果是 `[1, 1, 4, 2, 1, 1, 0, 0]`。

#### 示例 2：

```python
temperatures = [30, 40, 50, 60]
print(dailyTemperatures(temperatures))  # 输出 [1, 1, 1, 0]
```

**解释**：
- 每一天的温度都比前一天高，返回的结果是 `[1, 1, 1, 0]`。

#### 示例 3：

```python
temperatures = [50, 40, 30, 20, 10]
print(dailyTemperatures(temperatures))  # 输出 [0, 0, 0, 0, 0]
```

**解释**：
- 每一天的温度都在下降，返回的结果是 `[0, 0, 0, 0, 0]`，因为没有比当前温度更高的温度。

### **总结**

- 本题主要考察栈的应用，通过栈来帮助我们高效地找到每一天距离下一个更高温度的天数。
- 使用栈能够保证我们在一次遍历中完成计算，时间复杂度为 `O(n)`。
