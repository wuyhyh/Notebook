第240题是“搜索二维矩阵 II”（Search a 2D Matrix II），问题要求在一个 **升序排列的矩阵** 中查找目标值。矩阵的行和列都是升序排列的。

### 问题描述

给定一个 `m x n` 的矩阵 `matrix` 和一个目标值 `target`，返回 `True` 如果 `target` 存在于矩阵中，或者返回 `False` 如果它不存在。

- 每行的元素从左到右是升序排列的。
- 每列的元素从上到下是升序排列的。

### 示例

**示例 1:**

输入：
```plaintext
matrix = [
  [1,   4,  7, 11],
  [2,   5,  8, 12],
  [3,   6,  9, 16],
  [10, 13, 14, 17]
]
target = 5
```

输出：
```plaintext
True
```

**示例 2:**

输入：
```plaintext
matrix = [
  [1,   4,  7, 11],
  [2,   5,  8, 12],
  [3,   6,  9, 16],
  [10, 13, 14, 17]
]
target = 20
```

输出：
```plaintext
False
```

## 解题思路

### 思路：利用矩阵的有序特性

由于矩阵是 **升序排列** 的，且每行和每列都升序排列，因此可以利用以下策略进行高效搜索：

1. **从矩阵的右上角开始搜索**：
    - **如果目标值等于当前值**，则返回 `True`。
    - **如果目标值小于当前值**，则说明目标值不可能出现在当前列的下方，因为该列是升序排列的，因此可以 **排除当前列**（向左移动）。
    - **如果目标值大于当前值**，则说明目标值不可能出现在当前行的左侧，因为该行是升序排列的，因此可以 **排除当前行**（向下移动）。

2. 通过不断缩小搜索范围，最终找到目标值或确定目标值不存在。

### 代码实现：

```python
def searchMatrix(matrix, target):
    if not matrix:
        return False
    
    m, n = len(matrix), len(matrix[0])
    row, col = 0, n - 1  # 从右上角开始

    while row < m and col >= 0:
        if matrix[row][col] == target:
            return True
        elif matrix[row][col] > target:
            col -= 1  # 向左移动
        else:
            row += 1  # 向下移动
    
    return False
```

### 代码解释

1. **初始化**：
    - 获取矩阵的行数 `m` 和列数 `n`。
    - `row` 初始化为 0，表示从矩阵的第一行开始。
    - `col` 初始化为 `n - 1`，表示从矩阵的最后一列开始。

2. **循环遍历**：
    - `while row < m and col >= 0`：当行和列都在有效范围内时继续搜索。
    - **如果当前元素等于目标值**，则返回 `True`。
    - **如果当前元素大于目标值**，则目标值只能出现在当前列的左边，所以 `col -= 1`，向左移动一列。
    - **如果当前元素小于目标值**，则目标值只能出现在当前行的下方，所以 `row += 1`，向下移动一行。

3. **循环结束**：
    - 如果遍历完整个矩阵仍然没有找到目标值，返回 `False`。

### 时间复杂度和空间复杂度

- **时间复杂度**：`O(m + n)`，其中 `m` 是矩阵的行数，`n` 是矩阵的列数。我们从右上角开始，每次移动一行或者一列，最多需要 `m + n` 次操作。

- **空间复杂度**：`O(1)`，只用了常数的空间来存储索引变量 `row` 和 `col`。

## 总结

1. **从右上角开始搜索**，利用矩阵行和列的升序排列特性，进行高效的查找。
2. **时间复杂度为 `O(m + n)`**，比直接遍历整个矩阵的 `O(m * n)` 要高效得多。
