# C 语言基本概念

函数是用来构建程序的构建块。事实上，C程序就是函数的集合。

变量: 程序执行过程中一种临时存储数据的方法

每一个变量需要一种类型，类型会影响变量的存储方式以及能够在变量上执行的操作。

使用变量之前必须对其进行声明(为编译器所做的描述)，声明就是指定变量的类型和名字。

`int height = 8;`在变量声明中加入初始值，在这里数值`8`是一个初始化器。

在编写程序时，需要对变量、函数、宏和其他实体进行命名。这些名字称为标识符。

我们可以把C程序看成一连串的记号(token)，即许多在不改变意思情况下无法再分割的字符组。

`printf(fmt, exp1, exp1, ...)`这里的格式控制字符串包含普通字符和转换说明。跟随在`%`后面的信息指定了把数值从内部形式(
二进制)
转换成打印形式(字符)的方法。

`scanf()`本质上是一种模式匹配函数，试图把数组的字符组与转换说明相匹配。

赋值运算符要求它的左操作数必须是左值(lvalue)。左值表示对象，而不是常量或者计算的结果。

关系运算符在C语言表达式中产生的结果是0(假)或1(真)。

复合语句是由一对花括号，以及花括号内的声明和语句混合而成的。

C语言遵循的规则是`else`子句应该属于离它最近的且还未和其他`else`匹配的`if`语句。与Python缩进匹配要区分。

逗号表达式: `exp1, exp2`，第一步计算表达式1的值然后扔掉，第二补计算表达式2，把这个值作为整个表达式的值。我们可以把多个表达式粘在一起，但是不多见

`break`语句刚好把程序控制转移到循环体之后，`continue`语句刚好把程序控制转移到循环体末尾之前。花括号的两侧。
`continue`确实很少使用。

## 基本类型

默认情况下，C语言中的整型变量都是有符号的。

整型只有6种，随着编译器的不同，取值范围可能也不同。

```c++
short int               _a;
unsigned short int      _a;
int                     _a;
unsigned int            _a;
long int                _a;
unsigned long int       _a;
```

常量: 程序中以文本形式出现的数。

八进制和十六进制只是书写的形式，并不会对数的实际存储方式产生影响(整数都是以二进制形式存储的)

整数溢出的行为要根据操作数是有符号型还是无符号型来确定。有符号整数溢出，程序的行为是未定义的。无符号整数溢出时，结果是对2^n取模，也就是发生回绕。

```c++
unsigned char a = 255;
a = a + 1;
printf("%u\n", a);  // 输出 0，因为 256 % 256 == 0
```

读写无符号整数、短整数、长整数需要一些新的转换说明符。`%d`只适合`int`类型。

C语言将字符当成小整数进行处理。字符常量实际上是int类型而不是char类型。字符拥有和数相同的属性。

```c++
#include <stdio.h>

int main() 
{
    char c = 'A';     // 小整数 65
    printf("%d\n", c);      // 输出 65
    printf("%c\n", c + 1);  // 输出 'B'
    printf("%lu\n", sizeof('A'));  // 输出 4（int 类型）
    return 0;
}
```

字符常量在 C 语言中本质是 int 类型，而字符类型 char 是一种小的整数类型，因此字符和整数在很多地方是可以混用的。

C语言标准没有说明普通的char类型数据是有符号的还是无符号的。你最好区分。

当有符号整数和无符号整数混用的时候，有符号整数会被自动当成无符号整数的值。你最好避免这种情况的发生。

强制类型转换表达式用于显示那些肯定会发生的转换，我们可以控制编译器强制进行我们需要的转换。

`sizeof`运算符允许程序获取存储指定类型的值所需要的内存空间。

## 数组

标量(scalar)具有保存单一数据项的能力。

聚合变量(aggregate)可以存储成组的数值。C语言中有2种聚合类型: 数组(array)和结构(structure)

C语言不要求检查下标的范围。当下标超出范围时，程序可能执行不可预知的行为。

数组初始化器是一个用花括号括起来的常量表达式列表。在使用数组初始化器的时候可以省略数组的长度。可以使用方括号构成指示器初始化少量元素，其他元素默认为0；

运算符sizeof可以确定数组的大小(字节数)

C语言是按行主序来存储二维数组的。多维数组的角色较弱，因为我们可以只用指针数组来方便存放多维数据结构。

const在数组声明中是很常用的，程序经常会用到一些不需要修改的数据。

```c++
const char hex_chars[] = {'1','A'};
```

变长数组常见于main()以外的函数中。

## 函数

函数是C程序的构建块。

变量声明和语句可以混在一起，只要在变量第一次使用之前进行声明就行。
在调用一个函数之前，必须对其进行声明或定义。

在C语言中，实际参数是按值传递的。调用函数时，计算出每个实际参数的值并将其赋给相应的形式参数。在函数执行过程中对形式参数的改变并不会影响实际参数的值。

C语言没有给函数提供任何简便的方法来确定传递给他的数组的长度。如何函数需要，必须把长度作为额为的参数提供出来。

如果形式参数是多维数组，只能省略第一维的长度。

复合字面量是通过指定其包含的元素而建立的没有名字的数组。格式：先在一对圆括号内给定类型名，随后是一个初始化器，用来指定初始值。

```c++
total = sum_array((int []){1, 2, 3}, 3);
```

return语句与exit的差异是，不管哪个函数调用exit都会使程序终止，return语句只有才main函数中被调用时才会导致函数终止。

每种shell都有自己测试状态的方法，bash中变量`$?`包含上一个程序的运行状态。

函数可以交叉调用，这是一种间接递归的形式，是合法的。但是要确保两个函数都可以终止。

## 程序结构

当一个程序包含多个函数时所产生的问题。

函数体内声明的变量称为局部变量。

变量的存储期是指程序执行时，能够确保变量的存储空间必定存在的时间。

局部变量具有自动存储期，函数返回时收回。块作用域。

在局部变量声明中使用static可以使变量具有静态存储期，在整个程序执行期间都会保留变量的值。程序执行期间他占据的内存单元是不变的。
静态变量可以对其他函数隐藏数据，同时将来在同一个函数再调用时将保留这些数据。

外部变量是声明在任何函数体之外的。外部变量具有静态存储期和文件作用域。

作用域规则：程序块内部声明的变量会临时覆盖外部旧的声明。

对于单个文件的程序的一种组织方法:

    #include指令;
    #define指令;
    类型定义;
    外部变量的声明;
    除main()之外的函数的原型;
    main()函数的定义;
    其他函数的定义;

最好在每个函数定义之前放置盒型注释，描述函数的目的和参数等信息。

## 指针

计算机将内存分割为字节，一个字节存储8位的信息。每一个字节有唯一的地址。一个变量占用一个或多个字节的内存，把第一个字节的地址称为变量的地址。

指针就是地址，指针变量是存储地址的变量。

C语言要求每个指针变量只能指向一种特定类型的对象。

为了使用指针，C语言设计了一对特殊的运算符。为了获取变量的地址，使用取地址`&`运算符。为了获得对指针所指对象的访问，使用间接寻址
`*`运算符。
可以理解为数学上的逆运算。不要把间接寻址运算符用于没有初始化的指针变量，会导致未定义的行为。

如果两个指针变量具有相同的类型，可以使用赋值运算进行指针的复制。

使用const表面函数不会改变指针参数所指的对象：

```c++
void func(const int *p)
```

## 指针和数组

C语言允许对指针进行3种算术运算，加上一个整数，减去一个整数，两个指针相减。理解为array index。
指针算术运算的整数会根据指针的类型进行缩放。

只有当2个指针指向同一个数组的时候，减法或者使用关系运算符进行比较才有意义。

指向复合字面量的指针:

```c++
int *p = (int []){1, 2, 3};
```

最常见的

```c++
*p++ = j; // 把值存放到一个数组元素中，然后前进到下一个元素。
```

可以把数组名作为指针，但是不能给数组名赋新的值。

给函数传递数组的时间与数组的大小无关，因为并没有对数组进行复制。如果需要，可以把数组型形式参数声明为指针。

## 字符串

字符串常量和字符串变量。
字符串变量可以在执行的时候改变。

字面串是源文件的一部分，是程序中一段引号围起来的文本。
字面串经过编译之后生成字符串，字符串是位于存储器中的，以空字符结尾的字符序列。

本质上，C语言将字面串当做字符数组进行处理。不要混淆空字符和零字符。

通常情况下可以在任何C语言允许使用char*指针的地方使用字面串。

可以对字面串取下标：

```c++
char digit_to_hex_char(int digit)
{
    return "0123456789ABCDEF"[digit];
}
```

试图改变字面串会导致未定义的行为。

字面串与字符常量是不一样的。'a'与"a"是不一样的，一个是整数，一个是长度为2的数组。

字符串变量: C语言采取了不同的方式，只要保证字符串是以空字符结尾的，任何一维的字符数组都可以用来存储字符串。

字符串的长度取决于空字符的位置，而不是存放字符串的字符数组的长度。

字符数组与字符指针：

```c++
char date[] = "Apirl 3";
char *date = "Apirl 3";
```

这两者是不能互换的，声明为数组时，数组中的元素是可以改变的。声明为指针指向字面串的时候，字面串的内容是不能改变的。
第一个是date是一个数组名，第二个date是一个指针变量。

使用C语言的字符串库<string.h>

```c++
char *strcpy(char *s1, char *s2);
char *strncpy(char *s1, char *s2, sizeof(str1) - 1); // 确保str1可以装得下str2

size_t strlen(char *s); // 返回空字符前的字符个数作为字符串的长度，不包括空字符。

char *strcat(char *s1, char *s2);

char *strcmp(const char *s1, const char *s2); // 返回一个小于等于大于0的值
```

字符串数组与命令行参数。我们需要一种参差不齐的数组，数组中的每一个元素都是指向字符串的指针。

## 预处理器

预处理器会引入#include的内容，将#define更换为空行，注释会被换成空格。`gcc -E`可以得到预处理器的处理结果。

预处理指令主要是3种：宏定义、文件包含、条件编译

指令可以出现在程序中的任何地方。
指令通常在第一个换行符处结束，如果想在下一行延续指令，需要在末尾使用`\`字符。

宏定义中替换列表为空是合法的:

```c++
#define DEBUG // 控制条件编译
```

使用带参数的宏的时候，宏的名字和左括号之间必须没有空格。
带参数的宏经常用作简单的函数。

宏定义可以包含两个专用的运算符`#`和`##`
`#`运算符会将宏的第一个参数转换为字面串。
`##`运算符可以将两个记号粘接在一起，成为一个记号。注意不能嵌套使用。

宏定义的范围是文件作用域。#undef指令可以用来取消一个宏的定义，以便给出新的定义。

如果一个宏需要包含一系列的语句，那么可以只用`do { ... } while (0)`包含起来，确保只执行一次。

预定义宏可以告诉我们一些编译或者编译器的信息。
我们可以使用`__LINE__`和`__FILE__`来找到错误。

## 编写大型程序

常见的程序由多个源文件组成，其中一个源文件必须包含main()的定义，这是程序的起始点。还有一些头文件。
源文件包含函数的定义和外部变量。头文件包含了可以在源文件之间共享的信息。

`#include`指令告诉预处理器打开指定的文件，并且把此文件的内容插入当前文件中。
有两种书写方式，差异在于编译器定义头文件的方式。通常可以改变头文件的搜索方式，通过`-I`命令行选项。

共享的宏和类型定义应该放在头文件中。

函数和变量在共享的时候要注意声明和定义的区别。一定要确保变量的所有声明和定义的时候一致。
为了共享函数，要把函数的定义放在一个源文件中，然后再需要调用此函数的其他文件中放置声明。

```c++
int i; // 声明并且定义了变量i，编译器会留出空间
extern int i; // 声明但是不定义，编译器不会留出空间
```

构建多文件程序的时候需要编译和链接，这是两个独立的过程。
头文件的存在是为了为编译器而不是链接器提供信息。
链接器只会链接实际用到的函数。

必须对每个源文件进行编译，不需要编译头文件，因为头文件的内容在预处理的时候就被包含进源文件了。
链接的一个职责是解决编译器遗留的外部引用问题。

C语言编译器允许我们提供指定宏的值的方法，`-D`选项允许用命令行来指定宏的值。

```c++
extern int a[];
extern int *a;
```

在用于表达式时，数组会衰退为指针。但是在变量声明的时候，这是两个完全不同的东西。

## 结构 联合 枚举

结构是可能具有不同类型的值的集合。

联合和结构很类似，不同之处在于联合的成员共享同一存储空间。

枚举是一种整数类型，他的值由程序员来命名。

当需要存储相关数据项的集合的时候，结构是一种合乎逻辑的选择。
结构的成员在内存中是按声明的顺序存储的。
结构内部是一个名字空间。结构的成员的名称和外部的变量不会起冲突。

结构可以在声明的时候初始化，必须使用常量进行初始化。
结构也是支持指示器的，`.name = value`，顺序无所谓，名称必须正确。

对数组最频繁的操作是取下标，对结构来说就是选择成员。
访问成员运算符是`.`，具有最高的优先级。`&part.on_hand`的值是part.on_hand的地址。

结构的标记(tag)用了定义某种结构类型的名字。

```c++
struct part {
    int number;
    char name[NAME_LEN + 1];
    int on_hand;
```

这里的part是一个tag。结构标记只有在前面放置了单词struct时才会有意义。

复合字面量也可以用于结构。

数组和结构的组合没有限制，可以互相嵌套。
数组和结构最常见的组合就是元素为结构的数组，可以用作最简单的数据库。

联合也是由一个或者多个成员构成的，这些成员可能具有不同的类型。编译器只会为最大的成员分配足够的内存空间。
结构和联合唯一的不同是，结构的成员位于不同的地址中，联合的成员位于同一个地址。

联合有一个重要的应用：创建不同数据类型混合的数据结构。

枚举和结构、联合没有什么共同的地方，但是声明的方式很类似。
枚举变量的处理方式和整数一样，主要是用来记录一些少量有限的有具体意义的值。

## 指针的高级应用

动态存储分配和指向函数的指针。

```c++
void *malloc(size_t size);
void *cmalloc(size_t nmemb, size_t size); // 元素个数，每个元素的大小
void *realloc(void *ptr, size_t size); // 调整数组大小使他更适合需要
```

```c++
char *p = (char*)malloc(n + 1);
strcpy(p, "abc");
```

free(p)函数会释放p指向的内存块，但是不会改变p本身。这个时候再修改p指向的内存会导致严重的错误。

当函数的实际参数是指针变量的时候，有时候会希望函数能通过让指针指向别处来改变此变量。这就需要用到指向指针的指针。

指向函数的指针，将函数指针作为参数来传递是很常见的。C语言将指向函数的指针当做指向数据的指针对待。

受限指针`int * restrict p;`指针p指向的对象如果需要修改的话，只能通过p，这是唯一的方式。

```c++
struct vstring{
    int len;
    char chars[];
}
```

当结构的最后一个成员是数组的时候，其长度可以省略。
弹性数组成员的不同寻常之处在于，他在结构内并不占空间。具有弹性数组成员的结构是不完整类型。不完整类型缺少用于确定所需内存大小的信息。

```c++
int compare_ints(const void *p, const void *q)
{
    return *(int*)p - *(int*)q;
}

int compare_strings(const void *p, const void *q)
{
    return strcmp(*(char**)p, *(char**)q);
}
```
