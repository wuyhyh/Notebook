### **LeetCode 第20题 - 有效的括号 (Valid Parentheses)**

#### **题目描述**
给定一个只包括 `'('`, `')'`, `'{'`, `'}'`, `'['`, `']'` 的字符串，判断字符串是否有效。

有效的括号字符串需满足：
- 括号必须以正确的顺序闭合。
- 每个左括号必须有一个对应的右括号。

#### **示例**

示例 1:
```python
s = "()"
print(isValid(s))  # 输出 True
```

示例 2:
```python
s = "([)]"
print(isValid(s))  # 输出 False
```

示例 3:
```python
s = "{[]}"
print(isValid(s))  # 输出 True
```

#### **思路分析**

我们可以使用栈（Stack）来解决这个问题。栈是一种后进先出（LIFO）的数据结构，它特别适合处理这类需要匹配的括号问题。

**栈的基本操作**：
- **push**：压入栈中。
- **pop**：从栈顶弹出元素。

#### **解决步骤：**
1. 遍历字符串中的每个字符。
2. 如果当前字符是一个左括号 `(`、`[` 或 `{`，则将其压入栈中。
3. 如果当前字符是一个右括号 `)`、`]` 或 `}`：
    - 检查栈是否为空。如果栈为空，说明没有对应的左括号，返回 `False`。
    - 如果栈不为空，弹出栈顶元素并检查其是否与当前右括号匹配。如果不匹配，返回 `False`。
4. 最后，检查栈是否为空。如果栈为空，说明所有的括号都被正确匹配和关闭，返回 `True`；否则，返回 `False`。

### **代码实现**

```python
def isValid(s: str) -> bool:
    # 定义一个栈来保存左括号
    stack = []
    
    # 定义一个字典，映射右括号到对应的左括号
    bracket_map = {')': '(', '}': '{', ']': '['}
    
    # 遍历字符串中的每个字符
    for char in s:
        if char in bracket_map:
            # 如果是右括号，弹出栈顶的左括号
            top_element = stack.pop() if stack else '#'
            
            # 如果左括号与右括号不匹配，返回 False
            if bracket_map[char] != top_element:
                return False
        else:
            # 如果是左括号，压入栈中
            stack.append(char)
    
    # 最后，如果栈为空，说明括号匹配正确
    return not stack
```

### **代码解析**

1. **`bracket_map` 字典**：
    - 用来存储右括号和其对应的左括号之间的映射关系。
    - 比如 `')'` 对应 `'('`，`'}'` 对应 `'{'`，`']'` 对应 `'['`。

2. **遍历字符串**：
    - 如果当前字符是一个右括号（即存在于 `bracket_map` 中），我们就从栈中弹出一个元素，并检查它是否与当前右括号对应的左括号匹配。
    - 如果当前字符是一个左括号，我们将它压入栈中。

3. **检查栈**：
    - 如果在处理完所有字符后栈为空，说明所有的括号都成功匹配并闭合，返回 `True`。
    - 如果栈不为空，说明有未闭合的左括号，返回 `False`。

### **时间复杂度**

- **时间复杂度**：`O(n)`，其中 `n` 是字符串 `s` 的长度。我们只需遍历一次字符串，对于每个字符，进行栈的基本操作（入栈和出栈）都是常数时间操作。
- **空间复杂度**：`O(n)`，栈的空间使用最坏情况下为 `n`，即当字符串中的所有字符都是左括号时，栈的大小为 `n`。

### **示例**

#### 示例 1：

```python
s = "()"
print(isValid(s))  # 输出 True
```

**解释**：字符串中的左右括号正确配对，返回 `True`。

#### 示例 2：

```python
s = "([)]"
print(isValid(s))  # 输出 False
```

**解释**：左括号 `(` 和 `[` 是正确的，但右括号 `)` 与左括号 `[` 配对错误，因此返回 `False`。

#### 示例 3：

```python
s = "{[]}"
print(isValid(s))  # 输出 True
```

**解释**：字符串中的括号正确配对，返回 `True`。

#### 示例 4：

```python
s = "((()))"
print(isValid(s))  # 输出 True
```

**解释**：所有的括号都正确配对，返回 `True`。

### **总结**

- 使用栈可以高效地解决括号匹配问题，特别适合这类需要检查配对的场景。
- 时间复杂度为 `O(n)`，空间复杂度为 `O(n)`，非常适合处理中等规模的字符串。
