# 第3章 TCP协议详解

TCP 协议更靠近应用层，在应用程序中具有更强的可操作性，一些重要的 socket 选项都与 TCP 相关。

本章从四个方面来讨论 TCP 协议：

- TCP 头部信息
- TCP 状态转移过程，TCP 连接的任意一端都是一个状态机。
- TCP 数据流，TCP 数据流有两种类型，交互数据流和成块数据流
- TCP 数据流的控制，主要是两个方面，超时重传和拥塞控制

## 3.1 TCP 服务的特点

TCP 协议相对于 UDP 协议的特点是：面向连接、字节流和可靠传输。

TCP 协议的连接是一对一的，所以使用多播的应用程序不能使用 TCP 协议。
