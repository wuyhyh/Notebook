> 计算机的主要功能就是三个：计算、存储、输入输出。可以这样理解吗？

是的，**计算机的主要功能**可以概括为**计算、存储、输入输出**，这是对计算机操作的一个简洁但有效的理解。

### 1. **计算（Computation）**：

* **定义**：计算指的是计算机执行的数学运算和逻辑操作。现代计算机可以进行各种复杂的算术运算、逻辑推理、数据处理等。
* **硬件支持**：计算通常由\*\*中央处理器（CPU）\*\*完成，CPU负责执行各种指令，包括加法、减法、乘法、除法，以及更复杂的操作，如浮点运算、条件判断等。

### 2. **存储（Storage）**：

* **定义**：存储是指计算机保存数据和程序的能力。计算机通过不同类型的存储设备，管理和保存运行时的数据、操作系统、应用程序等信息。
* **硬件支持**：存储分为两大类：

    * **主存储（RAM）**：用于存放正在运行的程序和数据，是一种高速的临时存储。它是易失性的，一旦电源关闭，数据将丢失。
    * **辅助存储**：如硬盘（HDD）、固态硬盘（SSD）等，用于长期存储数据和文件。它是非易失性的，能在电源关闭后保存数据。

### 3. **输入输出（I/O）**：

* **定义**：输入输出指的是计算机与外部环境之间的数据交换过程。输入是指从外部接收数据，输出是指将数据传递给外部设备或用户。
* **硬件支持**：

    * **输入设备**：如键盘、鼠标、扫描仪等，用于输入数据到计算机。
    * **输出设备**：如显示器、打印机、扬声器等，用于显示或输出计算结果。

### 总结：

从功能上看，计算机确实可以归纳为这三个核心功能：

* **计算**：执行各种数学和逻辑操作；
* **存储**：保存数据和程序；
* **输入输出**：与外部环境进行数据交换。

这三个功能相互协作，共同支持计算机的运行和实现不同的任务。
