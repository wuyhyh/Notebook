LeetCode第78题是《子集》（Subsets），题目要求给定一个整数数组 `nums`，返回该数组所有可能的子集（幂集）。

---

### 题目描述

给定一个整数数组 `nums`，数组中的元素 **互不相同**，返回该数组的所有子集。

**注意**：子集中的元素可以按任意顺序排列，且输出结果中不能包含重复的子集。

---

#### 示例

**示例 1：**

```plaintext
输入: nums = [1,2,3]
输出: [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]
```

**示例 2：**

```plaintext
输入: nums = [0]
输出: [[],[0]]
```

---

### 解题思路

这个问题可以用 **回溯算法** 或 **迭代法** 来解决。

#### 方法 1：回溯法

通过回溯法，我们从一个空子集开始，每次对每个元素选择是否加入当前子集，递归地生成所有可能的子集。

#### 步骤

1. **递归函数设计**：
    - `backtrack(start, path)`：从 `start` 开始遍历数组 `nums`，`path` 表示当前子集。
    - 每次递归时，将当前子集 `path` 加入结果 `result` 中。
    - 对于当前索引 `i`，尝试将 `nums[i]` 加入子集，然后递归处理。

2. **递归终止条件**：
    - 遍历结束后，返回当前构造的子集。

---

### 回溯法代码实现（Python）

```python
def subsets(nums):
    def backtrack(start, path):
        # 将当前子集加入结果
        result.append(path[:])
        # 遍历剩余元素
        for i in range(start, len(nums)):
            path.append(nums[i])  # 选择当前元素
            backtrack(i + 1, path)  # 递归调用
            path.pop()  # 回溯，撤销选择

    result = []
    backtrack(0, [])
    return result
```

#### 时间复杂度
- **时间复杂度**：`O(2^n)`，因为数组的所有子集数量是 `2^n`，其中 `n` 是数组的长度。
- **空间复杂度**：`O(n)`，用于存储递归栈和当前路径。

---

#### 方法 2：迭代法

迭代法通过不断扩展已有子集来生成新子集。

#### 步骤

1. 初始化结果集为 `result = [[]]`，即包含一个空子集。
2. 遍历数组 `nums` 中的每个元素 `num`：
    - 对于当前结果集中的每个子集，将 `num` 添加进去生成新的子集。
3. 最后返回结果集。

---

### 迭代法代码实现（Python）

```python
def subsets(nums):
    result = [[]]
    for num in nums:
        # 为当前结果集中的每个子集添加 num
        result += [curr + [num] for curr in result]
    return result
```

#### 时间复杂度
- **时间复杂度**：`O(2^n)`，因为每次遍历都会扩展子集，子集数量为 `2^n`。
- **空间复杂度**：`O(2^n)`，结果集存储所有子集。

---

### 示例解析

以 `nums = [1, 2, 3]` 为例：

#### 回溯法
1. 初始：`path = []`，加入结果 `result = [[]]`
2. 选择 `1`：递归调用 `path = [1]`，结果更新为 `result = [[], [1]]`
3. 选择 `2`：递归调用 `path = [1, 2]`，结果更新为 `result = [[], [1], [1, 2]]`
4. 选择 `3`：递归调用 `path = [1, 2, 3]`，结果更新为 `result = [[], [1], [1, 2], [1, 2, 3]]`
5. 回溯到 `path = [1]`，选择 `3`，结果更新为 `result = [[], [1], [1, 2], [1, 2, 3], [1, 3]]`
6. 依次处理所有可能，直到遍历结束。

#### 迭代法
1. 初始：`result = [[]]`
2. 遍历 `1`：`result = [[], [1]]`
3. 遍历 `2`：`result = [[], [1], [2], [1, 2]]`
4. 遍历 `3`：`result = [[], [1], [2], [1, 2], [3], [1, 3], [2, 3], [1, 2, 3]]`

最终结果为：

```plaintext
[[], [1], [2], [1, 2], [3], [1, 3], [2, 3], [1, 2, 3]]
```

---

### 总结

- **回溯法**：更加直观，通过递归逐步构造子集。
- **迭代法**：利用已有子集扩展生成新子集，代码更加简洁。

两种方法都能高效解决子集问题，根据需求选择适合的实现方式即可。
