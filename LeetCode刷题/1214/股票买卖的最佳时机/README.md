### **LeetCode 第121题 - 买卖股票的最佳时机 (Best Time to Buy and Sell Stock)**

#### **题目描述**
给定一个数组 `prices` ，它的第 `i` 个元素是某支股票第 `i` 天的价格。

你可以完成 **最多一次** 买卖操作（即买一次卖一次），设计一个算法来计算你能获取的最大利润。返回最大利润。如果你不能获得利润，返回 `0`。

**注意**：
- 你必须在买入股票之前卖出股票。
- 如果没有利润可得，返回 `0`。

---

### **解题思路**

这是一个典型的股票买卖问题，目标是找到买入和卖出时机，使得利润最大化。

#### **思路一：暴力解法（双重循环）**
最直观的方法是使用两层循环，外层循环遍历买入的时间点，内层循环遍历卖出的时间点，对于每一对买入和卖出时间，计算利润，并记录最大的利润。

- **时间复杂度**：`O(n^2)`，因为我们需要两次遍历数组。
- **空间复杂度**：`O(1)`，只使用常数空间。

不过，这种方法的效率较低，在 `n` 很大的情况下会超时，因此需要优化。

#### **思路二：优化解法（一次遍历）**
优化的核心是注意到：
- 我们不需要再次遍历数组中所有的可能组合来计算最大利润。
- 我们只需要知道当前的最低买入价格，以及当前的最大利润。

**具体步骤**：
1. 初始化两个变量：`min_price`（记录到目前为止的最低价格）和 `max_profit`（记录最大利润）。
2. 遍历价格数组，对于每一天的价格：
    - 如果当天的价格小于 `min_price`，则更新 `min_price`。
    - 否则，计算今天卖出的利润（`prices[i] - min_price`），并更新 `max_profit`。
3. 返回 `max_profit`，即为最大利润。

#### **时间复杂度**：
- **时间复杂度**：`O(n)`，我们只需要遍历一次数组。
- **空间复杂度**：`O(1)`，只使用常数空间。

---

### **代码实现**

```python
class Solution:
    def maxProfit(self, prices: list[int]) -> int:
        # 初始化买入的最低价格为正无穷，最大利润为0
        min_price = float('inf')
        max_profit = 0
        
        for price in prices:
            # 更新最低买入价格
            if price < min_price:
                min_price = price
            # 计算当前卖出后的利润，并更新最大利润
            max_profit = max(max_profit, price - min_price)
        
        return max_profit
```

### **代码解析**
1. **初始化变量**：
    - `min_price = float('inf')`：将最低价格初始化为正无穷，因为我们需要找到最小值。
    - `max_profit = 0`：最大利润初始化为 `0`，表示最初没有交易。

2. **遍历价格**：
    - 每次遇到价格 `price`，我们更新 `min_price` 为当前价格与之前 `min_price` 中较小的那个。
    - 计算当前的利润 `price - min_price`，并更新 `max_profit` 为当前利润与之前 `max_profit` 中较大的那个。

3. **返回最大利润**：最终返回 `max_profit`。

---

### **示例测试**

#### 示例 1：
```python
prices = [7, 1, 5, 3, 6, 4]
solution = Solution()
print(solution.maxProfit(prices))  # 输出 5
```

**解释**：
- 最佳交易是在第2天买入（价格为 `1`），在第5天卖出（价格为 `6`），获得利润 `6 - 1 = 5`。

#### 示例 2：
```python
prices = [7, 6, 4, 3, 1]
solution = Solution()
print(solution.maxProfit(prices))  # 输出 0
```

**解释**：
- 在这个数组中，价格一直在下降，没有机会获得利润，因此返回 `0`。

---

### **总结**
- 这道题要求找到最佳买入和卖出时机，最大化利润。
- 我们可以通过遍历一次数组，记录当前的最低买入价格，并计算每一天卖出的利润，来找到最大利润。
- 通过这种优化后的方法，时间复杂度为 `O(n)`，空间复杂度为 `O(1)`，显著提高了效率。
