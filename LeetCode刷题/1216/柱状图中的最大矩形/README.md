### **LeetCode 第84题 - 柱状图中最大的矩形 (Largest Rectangle in Histogram)**

#### **题目描述**

给定一个整数数组 `heights`，其中 `heights[i]` 表示第 `i` 根柱子的高度。求在该柱状图中，能够形成的最大矩形的面积。

#### **示例**

示例 1:
```python
heights = [2, 1, 5, 6, 2, 3]
print(largestRectangleArea(heights))  # 输出 10
```

示例 2:
```python
heights = [2, 4]
print(largestRectangleArea(heights))  # 输出 4
```

#### **思路分析**

这道题是典型的 **单调栈** 问题，目的是求柱状图中的最大矩形面积。我们需要利用栈来高效地求解这个问题。

**核心思路**：
1. **暴力解法的优化**：
    - 暴力解法中，考虑每一对柱子之间的面积，时间复杂度为 `O(n^2)`，不适用于较大的数据规模。
    - 我们可以使用栈来优化，保持栈中的柱子按照高度递增的顺序，这样可以在遍历过程中高效地计算每个柱子的最大矩形面积。

2. **如何利用栈**：
    - 使用一个栈来存储柱子的索引。
    - 对于每个柱子，当它的高度小于栈顶柱子的高度时，我们可以计算栈顶柱子形成的矩形面积，因为栈顶柱子的矩形必然以当前柱子为右边界。
    - 继续弹出栈顶柱子，直到栈顶柱子小于当前柱子或者栈为空。
    - 在遍历过程中，栈中的每个柱子代表了一个可能的矩形的高度，我们需要在遍历完成后继续处理栈中剩余的柱子。

3. **计算面积的关键**：
    - 对于栈顶元素，我们可以计算它的矩形面积。矩形的宽度是从栈顶元素到栈中下一个元素（如果栈为空，则宽度是当前柱子的索引 `i`）。

### **解题步骤**

1. **初始化**：
    - 使用栈来保存柱子的索引。
    - 遍历柱子时，对于每个柱子，我们都检查它与栈顶柱子之间形成的矩形面积。
    - 最后，处理完所有柱子后，栈中仍然可能有一些柱子，这些柱子的矩形也需要计算。

2. **遍历每个柱子**：
    - 对于每个柱子，若它的高度小于栈顶柱子的高度，弹出栈顶元素，并计算该柱子为高的矩形面积，直到栈顶柱子的高度小于当前柱子。
    - 然后将当前柱子的索引压入栈中。

3. **计算最终面积**：
    - 在遍历结束后，栈中可能还剩下元素，需要继续计算剩余元素形成的矩形面积。

### **代码实现**

```python
def largestRectangleArea(heights):
    # 添加一个高度为0的柱子，以确保栈中所有元素都能被处理
    heights.append(0)
    stack = []
    max_area = 0
    
    for i in range(len(heights)):
        # 如果栈为空或当前柱子的高度大于栈顶柱子的高度，将当前柱子的索引压入栈
        while stack and heights[i] < heights[stack[-1]]:
            h = heights[stack.pop()]  # 弹出栈顶柱子，计算它的矩形面积
            w = i if not stack else i - stack[-1] - 1  # 如果栈为空，宽度为i，否则为当前索引和栈顶索引的差
            max_area = max(max_area, h * w)  # 更新最大面积
        stack.append(i)  # 将当前柱子的索引压入栈
    
    return max_area
```

### **代码解析**

1. **`heights.append(0)`**：
    - 在最后添加一个高度为 0 的柱子，确保栈中的每个元素都能被处理，这样做的目的是确保所有栈中的柱子都能计算出它们的矩形面积。

2. **`while stack and heights[i] < heights[stack[-1]]:`**：
    - 如果当前柱子的高度小于栈顶柱子的高度，说明当前柱子是栈中较高柱子的右边界，我们就可以计算栈顶柱子所能形成的矩形面积。

3. **`h = heights[stack.pop()]`**：
    - 弹出栈顶元素，表示我们找到一个较小的柱子，需要计算它的矩形面积。

4. **`w = i if not stack else i - stack[-1] - 1`**：
    - 计算宽度。若栈为空，说明弹出的柱子是从头开始的，宽度是当前柱子的索引 `i`。否则，宽度是当前柱子索引与栈顶元素之间的差。

5. **`max_area = max(max_area, h * w)`**：
    - 更新最大矩形面积。

6. **`stack.append(i)`**：
    - 将当前柱子的索引压入栈。

### **时间复杂度**

- **时间复杂度**：`O(n)`，每个柱子只会被压入栈一次和弹出栈一次，因此总时间复杂度为 `O(n)`。
- **空间复杂度**：`O(n)`，栈中最多存储 `n` 个元素。

### **示例**

#### 示例 1：

```python
heights = [2, 1, 5, 6, 2, 3]
print(largestRectangleArea(heights))  # 输出 10
```

**解释**：
- 最大的矩形是由高度为 `5` 和 `6` 的柱子形成的，宽度为 `2`，所以面积为 `10`。

#### 示例 2：

```python
heights = [2, 4]
print(largestRectangleArea(heights))  # 输出 4
```

**解释**：
- 最大的矩形是由高度为 `4` 的柱子形成的，宽度为 `1`，所以面积为 `4`。

### **总结**

- 本题通过单调栈的方式高效地解决了柱状图中最大矩形的面积问题。
- 栈帮助我们以 `O(n)` 时间复杂度完成了面积计算，这比暴力解法要高效得多。
