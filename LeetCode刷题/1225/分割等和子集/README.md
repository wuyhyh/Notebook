LeetCode 第 416 题是 **“分割等和子集”（Partition Equal Subset Sum）**，这是一道经典的动态规划问题。

### 问题描述：

给定一个**非空**的整数数组 `nums`，将该数组划分成两个子集，使得这两个子集的和相等。如果能划分成功，返回 `true`；否则返回 `false`。

### 思路：

这道题可以转化为一个 **子集和问题**，即：  
我们需要找到一个子集，使得它的元素和等于数组所有元素和的一半。如果能找到这样一个子集，那么剩下的元素的和自然也会是一样的。

### 步骤：
1. **计算总和**：计算数组中所有元素的和。如果总和是奇数，则不能将数组分成两个和相等的子集，直接返回 `false`。
2. **目标和**：如果总和是偶数，设目标和为 `sum/2`，即找一个子集的和等于目标和。
3. **动态规划**：使用一个布尔数组 `dp`，其中 `dp[i]` 表示是否能够从数组中选出若干个元素，使得它们的和为 `i`。
4. **初始化**：`dp[0] = true`，表示总和为 0 时总是可以通过选择空子集实现。
5. **更新状态**：对于数组中的每个元素，逆序更新 `dp` 数组，防止重复使用同一个元素。

### 动态规划实现：

```python
def canPartition(nums):
    total_sum = sum(nums)
    
    # 如果总和是奇数，无法分成两个相等的子集
    if total_sum % 2 != 0:
        return False
    
    # 目标和
    target = total_sum // 2
    
    # 创建动态规划数组，初始化 dp[0] = True，表示和为 0 可以通过选择空集来实现
    dp = [False] * (target + 1)
    dp[0] = True
    
    # 遍历每个数字
    for num in nums:
        # 逆序更新 dp 数组，防止重复使用当前数字
        for i in range(target, num - 1, -1):
            dp[i] = dp[i] or dp[i - num]
    
    # 如果 dp[target] 为 True，表示存在和为 target 的子集
    return dp[target]
```

### 解释：
1. **总和计算**：首先计算数组的总和 `total_sum`。
2. **奇偶检查**：如果 `total_sum` 是奇数，直接返回 `False`，因为无法划分为两个和相等的子集。
3. **动态规划数组**：我们使用 `dp` 数组来记录是否能够得到某个和。`dp[i]` 表示是否能够从数组的某些元素中选择出元素使其和为 `i`。
4. **状态转移**：对于每个数字，逆序更新 `dp` 数组，以保证每个数字只被使用一次。
5. **最终结果**：如果 `dp[target]` 为 `True`，表示我们能找到一个子集，其和等于目标和，从而可以将数组分成两个和相等的子集。

### 时间复杂度：
- **时间复杂度**：O(n * target)，其中 `n` 是数组的长度，`target` 是目标和。最坏情况下，我们需要遍历每个元素，并更新 `dp` 数组。
- **空间复杂度**：O(target)，我们只使用了一个一维的 `dp` 数组，大小为 `target + 1`。

### 示例：

```python
nums = [1, 5, 11, 5]
print(canPartition(nums))  # Output: True
```

在这个例子中，总和是 `22`，目标和是 `11`，我们可以将数组分为 `[1, 5, 5]` 和 `[11]`，两者的和都为 `11`，因此返回 `True`。
