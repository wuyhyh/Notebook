### **LeetCode 第347题 - 前 K 个高频元素 (Top K Frequent Elements)**

#### **题目描述**
给定一个非空的整数数组，返回其中出现频率前 `k` 高的元素。

**要求：**
- 你可以假设 `k` 的值是合理的，即 `k <= 数组中不同元素的个数`。
- 请注意，数组中可能存在重复的元素。

#### **思路分析**

我们可以分步进行解决：
1. **统计频率**：首先，我们需要统计数组中每个元素出现的频率。
2. **找出前 K 个高频元素**：接下来，我们需要根据频率从高到低找出前 `k` 个元素。

**考虑的主要方法**：
1. **使用哈希表**（字典）统计每个元素的频率。
2. **使用优先队列**（堆）来选取前 `k` 个频率最高的元素。堆可以让我们在 `O(n log k)` 时间复杂度内找出前 `k` 个最大频率元素。

### **解决方法：**
我们可以通过以下几步来实现：
1. **统计频率**：使用一个哈希表 `count`，存储每个元素出现的频率。
2. **使用堆排序**：
    - 我们可以使用 Python 内置的 `heapq` 来实现一个最小堆（最小堆适合我们找到频率前 `k` 的元素）。
    - 如果堆的大小超过 `k`，就移除堆顶元素。
3. **最终结果**：堆中的元素就是频率前 `k` 的元素，返回堆中的所有元素。

### **代码实现**

```python
import heapq
from collections import Counter

class Solution:
    def topKFrequent(self, nums: list[int], k: int) -> list[int]:
        # 1. 统计每个数字的频率
        count = Counter(nums)
        
        # 2. 使用堆（最小堆），堆中存储 (频率, 数字) 元组
        # heapq 默认是最小堆，所以我们用负数来模拟最大堆
        # 然后从堆中选出前 k 个频率最高的元素
        return [item[0] for item in heapq.nlargest(k, count.items(), key=lambda x: x[1])]
```

### **代码解析**

1. **`Counter(nums)`**：
    - 我们使用 `Counter` 来统计数组中每个元素出现的次数，返回一个字典类型的对象，键是元素，值是元素的出现次数。

2. **`heapq.nlargest(k, count.items(), key=lambda x: x[1])`**：
    - `heapq.nlargest` 是一个可以高效获取堆中前 `k` 个最大元素的函数。
    - `count.items()` 返回的是一个元组列表，每个元组是 `(元素, 频率)`，我们根据频率（`x[1]`）对其进行排序，找出前 `k` 个频率最高的元素。

3. **`[item[0] for item in heapq.nlargest(k, count.items(), key=lambda x: x[1])]`**：
    - 通过列表推导式从每个元组中提取出元素（`item[0]`），构建最终的返回结果。

### **时间复杂度**

- **时间复杂度**：
    - 统计频率的时间复杂度是 `O(n)`，其中 `n` 是数组 `nums` 的长度。
    - 使用 `heapq.nlargest` 获取 `k` 个最频繁的元素，时间复杂度是 `O(k log n)`。这是因为在获取前 `k` 个元素时，`heapq` 内部会对 `n` 个元素进行排序操作。
    - 因此，总时间复杂度是 `O(n + k log n)`。

- **空间复杂度**：
    - 我们需要存储 `n` 个元素的频率信息，所以空间复杂度是 `O(n)`。

### **示例**

#### 示例 1：
```python
nums = [1,1,1,2,2,3]
k = 2
solution = Solution()
print(solution.topKFrequent(nums, k))  # 输出 [1, 2]
```

**解释**：
- `1` 出现了 3 次，`2` 出现了 2 次，`3` 出现了 1 次，返回前 `2` 个频率最高的元素 `[1, 2]`。

#### 示例 2：
```python
nums = [1]
k = 1
solution = Solution()
print(solution.topKFrequent(nums, k))  # 输出 [1]
```

**解释**：
- `1` 出现了 1 次，返回前 `1` 个频率最高的元素 `[1]`。

### **总结**

- **时间复杂度**：`O(n + k log n)`，我们需要先统计频率（`O(n)`），然后使用堆来获取前 `k` 个最大频率的元素（`O(k log n)`）。
- **空间复杂度**：`O(n)`，我们需要存储 `n` 个元素的频率信息。
- 使用 **堆**（最小堆）和 **计数器** 是高效且简洁的解决方案，适用于大规模数据集。
