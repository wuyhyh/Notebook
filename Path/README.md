## 能力建设

### Linux 内核高级研发工程师

- 精通 Linux 内核，至少阅读2-3个主要模块(调度，文件系统，网络，I/O，内存管理等)的源代码；
- 熟悉 Linux 平台上的 C 语言编程，熟悉多进程多线程编程，熟悉 socket 编程；
- 具有复杂系统软件的设计、开发和调优能力；
- 有丰富内核故障调试或内核社区补丁提交经验者优先；
- 有多平台（x86/ARM/RISC-V）内核与系统研发经验者优先；
- 了解主流虚拟化技术（XEN/KVM等）的实现，阅读过相关源代码者优先；

### Linux 系统运维工程师

- 熟悉操作系统架构设计和核心组件，熟悉网络、调度、IO等核心组件参数优化；
- 具备良好的问题分析和解决能力，能够跨方向协调上下游排查，处理复杂的操作系统问题；
- 有开源 Linux 社区经验者优先，熟悉 CentOS, Ubuntu 等主流操作系统；
