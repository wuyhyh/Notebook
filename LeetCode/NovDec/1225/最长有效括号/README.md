LeetCode 第 32 题是 **“最长有效括号”（Longest Valid Parentheses）**，这是一个经典的栈和动态规划问题。

### 问题描述：

给定一个由 `'('` 和 `')'` 组成的字符串，找出其中最长的有效括号子串的长度。

#### 示例：

输入: `"(()"`
输出: `2`
解释: 最长有效括号子串是 `"()"`，长度为 2。

输入: `")()())"`
输出: `4`
解释: 最长有效括号子串是 `"()()"`，长度为 4。

### 思路：

这道题的核心是 **找出有效括号子串**，有效括号指的是一对一对匹配的括号。我们可以使用以下几种方法来解决：

### 方法 1：栈法

通过栈来跟踪括号的位置，栈的核心思想是匹配括号时，弹出栈顶元素并计算长度。

#### 步骤：
1. **使用栈**：我们使用一个栈来存储括号的下标。栈顶元素表示当前未匹配的左括号 `'('` 或是已经匹配的右括号 `')'` 的索引。
2. **初始状态**：栈初始化时，可以将 `-1` 放入栈中，这样可以帮助计算以某个索引结束的有效括号的长度。
3. **遍历字符串**：
    - 对于每个字符：
        - 如果是左括号 `'('`，将其索引压入栈。
        - 如果是右括号 `')'`：
            - 弹出栈顶元素（表示匹配了一个左括号）。
            - 如果栈空了，表示当前右括号无法匹配任何左括号，将当前右括号的索引压入栈中。
            - 如果栈不空了，计算当前有效括号子串的长度，并更新最大长度。

#### 代码实现：

```python
def longestValidParentheses(s: str) -> int:
    stack = [-1]  # 栈初始化，加入 -1 帮助计算有效括号子串长度
    max_len = 0

    for i, char in enumerate(s):
        if char == '(':
            stack.append(i)  # 左括号入栈
        else:
            stack.pop()  # 右括号出栈
            if not stack:
                stack.append(i)  # 如果栈为空，表示当前右括号无法匹配，加入其索引
            else:
                max_len = max(max_len, i - stack[-1])  # 计算有效括号长度

    return max_len
```

### 解释：
1. **栈初始化**：我们初始化栈为 `[-1]`，这是为了处理字符串从索引 `0` 开始的有效括号子串。
2. **遍历字符串**：
    - 每遇到一个左括号 `'('`，我们将其索引压入栈。
    - 每遇到一个右括号 `')'`，我们弹出栈顶元素，表示当前的右括号与栈顶的左括号匹配。如果栈空了，说明当前右括号无法匹配任何左括号，所以我们将当前的右括号的索引压入栈。否则，我们计算当前有效括号子串的长度（`i - stack[-1]`），并更新最大长度。
3. **返回结果**：遍历结束后，`max_len` 存储了最长有效括号子串的长度。

### 时间复杂度：
- **时间复杂度**：O(n)，其中 `n` 是字符串的长度。我们遍历字符串一次，栈的操作（入栈和出栈）是常数时间操作。
- **空间复杂度**：O(n)，栈最多存储字符串中的所有索引。

### 示例：

```python
s = "(()"
print(longestValidParentheses(s))  # Output: 2

s = ")()())"
print(longestValidParentheses(s))  # Output: 4
```

### 方法 2：动态规划法

动态规划也可以解决这道题，核心思路是维护一个 `dp` 数组，`dp[i]` 表示以索引 `i` 结尾的最长有效括号的长度。

#### 步骤：
1. **状态转移**：
    - 如果 `s[i]` 是右括号 `')'`，我们检查 `s[i-1]` 是否是左括号 `'('`，如果是匹配的，则 `dp[i] = dp[i-2] + 2`。
    - 如果 `s[i-1]` 不是 `'('`，则我们检查 `dp[i-1]` 表示的有效括号是否能形成一个更大的有效括号。
2. **更新结果**：在遍历过程中，不断更新最大有效括号长度。

#### 代码实现：

```python
def longestValidParentheses(s: str) -> int:
    n = len(s)
    dp = [0] * n
    max_len = 0
    
    for i in range(1, n):
        if s[i] == ')':
            if s[i-1] == '(':  # 匹配
                dp[i] = (dp[i-2] if i >= 2 else 0) + 2
            elif i - dp[i-1] - 1 >= 0 and s[i - dp[i-1] - 1] == '(':  # 长度大于零且前面有左括号
                dp[i] = dp[i-1] + (dp[i - dp[i-1] - 2] if i - dp[i-1] - 2 >= 0 else 0) + 2
            max_len = max(max_len, dp[i])
    
    return max_len
```

### 总结：
- **栈法**是最常用的解法，时间复杂度 O(n)，并且代码简洁，易于理解。
- **动态规划法**虽然也可以解决问题，但在空间复杂度上相对较高，并且需要一些额外的状态转移逻辑。
