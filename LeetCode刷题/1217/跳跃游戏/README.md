LeetCode第55题是《跳跃游戏》（Jump Game），题目要求判断给定的一组非负整数数组是否能从数组的第一个元素出发，到达数组的最后一个元素。

### 题目描述
给定一个非负整数数组 `nums`，你最初位于数组的第一个位置。每个元素代表你在该位置可以跳跃的最大长度，判断你是否能够到达最后一个位置。

### 示例
```plaintext
输入: nums = [2,3,1,1,4]
输出: true
解释: 从第一个位置出发，你可以跳跃到达最后一个位置。
```

### 解题思路

这个问题可以通过贪心算法来解决。具体思路是从数组的第一个元素开始，不断更新当前能够跳到的最远位置。如果某一时刻最远位置已经超过或等于数组的最后一个位置，则返回 `true`，否则返回 `false`。

### 贪心算法步骤
1. 初始化一个变量 `farthest`，表示当前能到达的最远位置。
2. 遍历数组，检查当前元素 `i` 是否超出了已经能够到达的最远位置 `farthest`。如果是，说明不能再继续跳跃，直接返回 `false`。
3. 否则，更新 `farthest` 为 `max(farthest, i + nums[i])`，即当前能跳到的位置与当前位置跳跃的最大距离之和。
4. 如果在遍历过程中，`farthest` 超过或等于数组的最后一个位置，则返回 `true`。

### 代码实现（Python）
```python
def canJump(nums):
    farthest = 0
    for i in range(len(nums)):
        if i > farthest:  # 如果当前索引超出了能够到达的最远位置，返回 False
            return False
        farthest = max(farthest, i + nums[i])  # 更新最远可达位置
        if farthest >= len(nums) - 1:  # 如果最远可达位置已经到达或超过最后一个位置
            return True
    return False
```

### 时间复杂度
- 时间复杂度是 `O(n)`，其中 `n` 是数组 `nums` 的长度。我们只需要遍历一次数组。

### 空间复杂度
- 空间复杂度是 `O(1)`，只使用了常数空间来存储变量。

这样我们就能判断是否能跳到最后一个位置了！
