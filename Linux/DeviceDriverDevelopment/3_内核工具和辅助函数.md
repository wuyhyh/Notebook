## 3.1 理解宏`container_of`

container_of 主要用在内核通用容器中。

```c++
container_of(pointer, container_type, container_filed);
```

## 3.2 链表

内核开发者只实现了双向循环链表，这个结构可以进一步实现 FIFO 和 LIFO。

## 3.3 内核睡眠机制

内核调度器管理要运行的任务列表，称为运行队列。

睡眠进程不再被调度，他们已经从运行队列中移除。
