第2题是“两数相加”（Add Two Numbers），要求将两个非空链表表示的非负整数相加，并返回一个新的链表表示结果。

### 问题描述

给定两个非空链表表示两个非负整数。其中，数字是逆序存储的，每个节点包含一个数字。您需要将这两个数字相加，并返回一个新的链表来表示它们的和。

### 示例

**示例 1:**

输入：
```plaintext
l1 = [2, 4, 3], l2 = [5, 6, 4]
```

输出：
```plaintext
[7, 0, 8]
```

解释：数字 342 + 465 = 807。因此，返回链表 [7, 0, 8]，表示 807。

**示例 2:**

输入：
```plaintext
l1 = [0], l2 = [0]
```

输出：
```plaintext
[0]
```

**示例 3:**

输入：
```plaintext
l1 = [9, 9, 9, 9, 9, 9, 9], l2 = [9, 9, 9, 9]
```

输出：
```plaintext
[8, 9, 9, 9, 0, 0, 0, 1]
```

### 解题思路

我们可以模拟加法的过程，从两个链表的头部开始，逐位相加，并处理进位（carry）。具体步骤如下：

1. **初始化指针和进位**：
    - 创建一个虚拟头节点 `dummy`，用于构建结果链表。
    - 初始化指针 `current`，用于遍历结果链表。
    - 初始化进位 `carry` 为 0，表示当前的进位。

2. **遍历两个链表**：
    - 同时遍历两个链表，逐位相加。如果当前链表有节点，就取出当前节点的值；如果没有节点，则默认为 0。
    - 将两个链表当前位的和与进位相加，计算当前位的结果和新的进位。
    - 将当前位的结果存储到结果链表中，并更新进位。

3. **处理最终的进位**：
    - 如果遍历完两个链表后还有进位，就将进位添加到结果链表中。

4. **返回结果链表**：
    - 返回虚拟头节点 `dummy` 的下一个节点，这就是合并后的链表。

### 代码实现

```python
# Definition for singly-linked list.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def addTwoNumbers(l1: ListNode, l2: ListNode) -> ListNode:
    # 创建一个虚拟头节点
    dummy = ListNode()
    current = dummy
    carry = 0
    
    # 遍历两个链表
    while l1 or l2 or carry:
        # 获取当前位的值，若链表为空，则值为0
        val1 = l1.val if l1 else 0
        val2 = l2.val if l2 else 0
        
        # 计算当前位的和以及新的进位
        total = val1 + val2 + carry
        carry = total // 10  # 更新进位
        current.next = ListNode(total % 10)  # 当前位的值
        current = current.next
        
        # 移动指针
        if l1:
            l1 = l1.next
        if l2:
            l2 = l2.next
    
    return dummy.next
```

### 代码解释

1. **虚拟头节点**：
    - 我们使用一个虚拟节点 `dummy` 来简化链表的构建过程。
    - `current` 指针指向当前链表的末尾，负责将计算结果追加到链表中。

2. **遍历链表**：
    - 在 `while l1 or l2 or carry` 循环中，我们不断地从两个链表取出当前位的值，如果当前链表为空，则默认为 0。
    - 计算当前位的和（包括进位），然后更新进位 `carry` 和当前位的结果（`total % 10`）。

3. **处理进位**：
    - 如果最后还有进位（`carry` 不为 0），就将进位作为新的节点添加到结果链表的末尾。

4. **返回结果**：
    - 最终返回 `dummy.next`，因为 `dummy` 是虚拟头节点，实际的结果链表从 `dummy.next` 开始。

### 时间复杂度和空间复杂度

- **时间复杂度**：`O(max(n, m))`，其中 `n` 和 `m` 分别是两个链表 `l1` 和 `l2` 的长度。我们最多需要遍历两个链表一次。
- **空间复杂度**：`O(max(n, m))`，我们需要创建一个新的链表来存储结果。

### 总结

- **模拟加法**：从两个链表的头节点开始，逐位相加，并处理进位。
- **处理进位**：如果加法结果超过 9，产生进位，需要将进位传递到下一位。
- **时间复杂度**：`O(max(n, m))`，空间复杂度 `O(max(n, m))`，这是最优解。
