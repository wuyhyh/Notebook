LeetCode 第 72 题是 **“编辑距离”（Edit Distance）**，这是一道经典的动态规划问题，主要考察字符串的变换和距离计算。

### 问题描述：

给定两个单词 `word1` 和 `word2`，计算将 `word1` 转换成 `word2` 所使用的最少操作数。允许的操作有：

- 插入一个字符
- 删除一个字符
- 替换一个字符

### 思路：

这个问题可以通过 **动态规划（Dynamic Programming）** 来解决。定义 `dp[i][j]` 为将 `word1` 的前 `i` 个字符转换成 `word2` 的前 `j` 个字符所需的最少操作数。

#### 动态规划的转移方程：
1. **初始条件**：
    - `dp[0][0] = 0`，表示空字符串转换为空字符串，不需要任何操作。
    - `dp[i][0] = i`，表示 `word1` 的前 `i` 个字符转换为空字符串，需要 `i` 次删除操作。
    - `dp[0][j] = j`，表示空字符串转换成 `word2` 的前 `j` 个字符，需要 `j` 次插入操作。

2. **状态转移**：
    - **如果字符相同**，即 `word1[i-1] == word2[j-1]`，则无需进行任何操作，`dp[i][j] = dp[i-1][j-1]`。
    - **如果字符不同**，我们有三种操作选择：
        - **插入**：将字符 `word2[j-1]` 插入到 `word1[i-1]` 后面，转移为 `dp[i][j-1] + 1`。
        - **删除**：删除字符 `word1[i-1]`，转移为 `dp[i-1][j] + 1`。
        - **替换**：将字符 `word1[i-1]` 替换为 `word2[j-1]`，转移为 `dp[i-1][j-1] + 1`。
    - 以上三种操作中，选择最小的作为 `dp[i][j]` 的值。

### 代码实现：

```python
def minDistance(word1: str, word2: str) -> int:
    m, n = len(word1), len(word2)
    
    # 创建 dp 数组，dp[i][j] 表示 word1 的前 i 个字符转化为 word2 的前 j 个字符的最小编辑距离
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    # 初始化第一列和第一行
    for i in range(m + 1):
        dp[i][0] = i  # word1 的前 i 个字符转化为空字符串需要 i 次删除
    for j in range(n + 1):
        dp[0][j] = j  # 空字符串转化为 word2 的前 j 个字符需要 j 次插入
    
    # 填充 dp 数组
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i - 1] == word2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]  # 如果字符相同，不需要操作
            else:
                dp[i][j] = min(
                    dp[i - 1][j] + 1,  # 删除操作
                    dp[i][j - 1] + 1,  # 插入操作
                    dp[i - 1][j - 1] + 1  # 替换操作
                )
    
    # 最终结果是将 word1 转换为 word2 的编辑距离
    return dp[m][n]
```

### 解释：
1. **初始化**：
    - 创建一个二维 `dp` 数组，`dp[i][j]` 表示将 `word1` 的前 `i` 个字符转换成 `word2` 的前 `j` 个字符所需的最少操作数。
    - 初始化 `dp[0][j]` 为 `j`，表示将空字符串转化为 `word2` 的前 `j` 个字符需要 `j` 次插入。
    - 初始化 `dp[i][0]` 为 `i`，表示将 `word1` 的前 `i` 个字符转化为空字符串需要 `i` 次删除。

2. **状态转移**：
    - 如果 `word1[i-1] == word2[j-1]`，则 `dp[i][j] = dp[i-1][j-1]`，表示不需要任何操作。
    - 如果不相等，选择插入、删除、替换中最小的操作次数。

3. **返回值**：
    - 最终，`dp[m][n]` 就是将整个 `word1` 转换为整个 `word2` 所需的最少操作数。

### 时间复杂度：
- **时间复杂度**：O(m * n)，其中 `m` 是 `word1` 的长度，`n` 是 `word2` 的长度。我们需要遍历整个 `dp` 数组来计算最小编辑距离。
- **空间复杂度**：O(m * n)，我们使用了一个二维数组 `dp` 来存储中间结果。

### 示例：

```python
word1 = "horse"
word2 = "ros"
print(minDistance(word1, word2))  # Output: 3
```

解释：将 `"horse"` 转换为 `"ros"` 的最小操作为：
1. `"horse"` → `"orse"`（删除 `'h'`）
2. `"orse"` → `"ose"`（删除 `'r'`）
3. `"ose"` → `"ros"`（替换 `'e'` 为 `'r'`）

所以返回 `3`。
