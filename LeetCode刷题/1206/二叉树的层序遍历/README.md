LeetCode 第102题：“二叉树的层序遍历”（**Binary Tree Level Order Traversal**）要求我们返回一个二叉树的**层序遍历**结果。层序遍历（BFS，广度优先遍历）是按从上到下、从左到右的顺序逐层遍历树的节点。

### 思路：
1. **层序遍历**的基本思想是：
    - 使用队列（Queue）来实现广度优先遍历。我们从根节点开始，把根节点加入队列。
    - 然后反复从队列中取出节点，并把它的左右子节点依次加入队列。
    - 每一层的节点可以放在一个新的列表中，逐层存储结果。

2. **步骤**：
    - 使用队列存储当前层的节点。
    - 每次处理一个节点时，将它的左右子节点加入队列，并记录该层的节点值。
    - 继续处理队列中的节点直到队列为空，表示遍历完成。

### 代码实现：

```python
from collections import deque

class Solution(object):
    def levelOrder(self, root):
        """
        :type root: TreeNode
        :rtype: List[List[int]]
        """
        if not root:
            return []  # 如果树为空，返回空列表
        
        result = []  # 用来存储每一层的节点值
        queue = deque([root])  # 队列初始化，存储当前层的节点
        
        while queue:
            level_values = []  # 当前层的节点值
            level_size = len(queue)  # 当前层的节点数量
            
            # 逐个处理当前层的节点
            for _ in range(level_size):
                node = queue.popleft()  # 弹出队列中的一个节点
                level_values.append(node.val)  # 将当前节点的值加入当前层的列表
                
                # 如果当前节点有左子节点，加入队列
                if node.left:
                    queue.append(node.left)
                # 如果当前节点有右子节点，加入队列
                if node.right:
                    queue.append(node.right)
            
            result.append(level_values)  # 将当前层的节点值加入结果列表
        
        return result
```

### 解释：
1. **`levelOrder`** 方法：
    - 首先判断树是否为空，如果为空，直接返回空列表。
    - 然后使用队列（`deque`）来存储每一层的节点。队列首先包含根节点。
    - 在每次循环中，首先计算当前层的节点数量（`level_size`），然后逐一处理当前层的每个节点，将它的左右子节点（如果存在）加入队列，并将节点的值加入当前层的结果列表 `level_values`。
    - 最后，将 `level_values` 加入到 `result` 中，继续处理下一个层级的节点。

2. **队列的工作方式**：
    - 每次从队列中取出节点，并将它的左右子节点（如果存在）加入队列。
    - 每次处理一整层后，将这一层的结果添加到最终的结果列表中。

### 示例：

对于以下二叉树：
```
       3
      / \
     9  20
        /  \
       15   7
```

- **第一层**（根节点）：`[3]`
- **第二层**：`[9, 20]`
- **第三层**：`[15, 7]`

输出结果是：
```python
[[3], [9, 20], [15, 7]]
```

### 时间复杂度：
- **时间复杂度**：\(O(n)\)，其中 `n` 是树的节点数。每个节点被访问一次。
- **空间复杂度**：\(O(n)\)，最坏情况下（完全二叉树），队列会存储树的一层节点，空间复杂度为 \(O(n)\)。

### 总结：
- **队列实现层序遍历**：通过队列逐层处理每个节点的子节点，逐层记录节点值。
- **简洁有效**：时间复杂度和空间复杂度均为 \(O(n)\)，适用于任何二叉树的层序遍历。
