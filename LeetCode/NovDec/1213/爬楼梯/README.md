### **LeetCode 第70题 - 爬楼梯 (Climbing Stairs)**

#### **题目描述**
假设你正在爬楼梯。楼梯有 `n` 阶，每次你可以爬 1 或 2 步。你有多少种不同的方法可以爬到楼顶？

#### **解题思路**

这道题可以用 **动态规划**（Dynamic Programming）解决。假设我们在楼梯的第 `i` 阶，问题就变成了我们如何到达第 `i` 阶。显然，如果我们在第 `i-1` 阶，走一步就可以到达第 `i` 阶；如果我们在第 `i-2` 阶，走两步也可以到达第 `i` 阶。因此，到达第 `i` 阶的不同方法数是到达第 `i-1` 阶和第 `i-2` 阶的不同方法数之和。

#### **状态转移方程**：
设 `dp[i]` 表示到达第 `i` 阶楼梯的方法数，则有：
\[
dp[i] = dp[i-1] + dp[i-2]
\]
其中：
- `dp[i-1]` 表示从第 `i-1` 阶到第 `i` 阶。
- `dp[i-2]` 表示从第 `i-2` 阶到第 `i` 阶。

#### **边界条件**：
- `dp[0] = 1`：代表一种方法，不爬楼梯。
- `dp[1] = 1`：只有一种方法，到达第一阶楼梯，即直接爬一步。

#### **目标**：
求出 `dp[n]`，即到达第 `n` 阶楼梯的方法数。

---

### **代码实现**

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        # 边界条件
        if n == 1:
            return 1
        
        # 初始化 dp 数组
        dp = [0] * (n + 1)
        dp[0] = 1  # 到达第0阶楼梯的方法数是1（不爬楼梯）
        dp[1] = 1  # 到达第1阶楼梯的方法数是1（爬一步）

        # 填充 dp 数组
        for i in range(2, n + 1):
            dp[i] = dp[i - 1] + dp[i - 2]
        
        # 返回 dp[n]，即到达第 n 阶楼梯的方法数
        return dp[n]
```

### **代码解析**
1. **初始化 dp 数组**：我们用一个长度为 `n+1` 的数组 `dp` 来保存到达每一阶楼梯的方法数。初始化 `dp[0] = 1` 和 `dp[1] = 1`。
2. **填充 dp 数组**：从 `i = 2` 开始，根据状态转移方程 `dp[i] = dp[i-1] + dp[i-2]` 逐步填充 `dp` 数组，直到 `dp[n]`。
3. **返回结果**：最后，返回 `dp[n]` 即为到达第 `n` 阶楼梯的方法数。

---

### **优化空间复杂度**

由于 `dp[i]` 只依赖于 `dp[i-1]` 和 `dp[i-2]`，我们实际上可以使用两个变量来存储这两个值，而不需要使用整个 `dp` 数组。这将把空间复杂度从 `O(n)` 降到 `O(1)`。

#### **优化后的代码**：

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n == 1:
            return 1
        
        prev2, prev1 = 1, 1  # prev2 为 dp[i-2], prev1 为 dp[i-1]
        
        for i in range(2, n + 1):
            current = prev1 + prev2  # 当前阶梯的路径数
            prev2 = prev1  # 更新 prev2 为 prev1
            prev1 = current  # 更新 prev1 为当前的路径数
        
        return prev1
```

### **优化后的时间与空间复杂度**
- **时间复杂度**：`O(n)`，我们只需要遍历一遍从 `2` 到 `n` 的楼梯。
- **空间复杂度**：`O(1)`，我们只使用常数空间来存储当前和前两个阶梯的路径数。

---

### **示例测试**

#### 示例 1：
```python
n = 3
solution = Solution()
print(solution.climbStairs(n))  # 输出 3
```

**解释**：
- 第3阶楼梯可以通过以下三种方式到达：`1 + 1 + 1`，`1 + 2`，`2 + 1`。

#### 示例 2：
```python
n = 5
solution = Solution()
print(solution.climbStairs(n))  # 输出 8
```

**解释**：
- 第5阶楼梯可以通过以下八种方式到达。

### **总结**

- 本题是一个经典的动态规划问题，利用前两步的路径数来计算当前步的路径数。
- 通过优化，我们可以将空间复杂度从 `O(n)` 降到 `O(1)`，进一步提高了算法的效率。
- 该问题的关键是识别出状态转移方程：`dp[i] = dp[i-1] + dp[i-2]`，并理解如何通过动态规划来逐步解决问题。
